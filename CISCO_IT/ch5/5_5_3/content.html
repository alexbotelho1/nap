5.5 Navegação na GUI (Windows)  
  5.5.3 Exploração de ferramentas administrativas  

Gerenciador de dispositivos
O Gerenciador de dispositivos, como mostra a Figura 1, permite que você exiba todas as configurações dos dispositivos no computador. Uma tarefa comum dos técnicos é verificar os valores atribuídos para o IRQ, endereço de entrada/saída e a configuração de DMA dos dispositivos do computador. Para exibir os recursos do sistema no Gerenciador de dispositivos, use o seguinte caminho:

Iniciar > Painel de controle > Sistema > Hardware > Gerenciador de dispositivos > Exibir > Recursos

Gerenciador de tarefas
O Gerenciador de tarefas, na Figura 2, permite que você veja todos os aplicativos em execução e feche os que pararam de responder. Com o Gerenciador de tarefas, você pode monitorar o desempenho da CPU e da memória virtual, exibir todos os processos em execução e verificar informações sobre as conexões de rede. Para exibir informações no Gerenciador de tarefas, use o seguinte caminho:

CTRL-ALT-DEL > Gerenciador de tarefas

Visualizador de eventos
O Visualizador de eventos, na Figura 3, registra um histórico dos eventos relacionados aos aplicativos, à segurança e ao sistema. Esses arquivos de registro são uma valiosa ferramenta de solução de problemas. Para acessar o Visualizador de eventos, use o seguinte caminho:

Iniciar > Painel de controle > Ferramentas administrativas > Visualizador de eventos

Área de trabalho remota
A Área de trabalho remota permite que o computador assuma o controle remoto de outro. Esse recurso de solução de problemas só está disponível no Windows XP Professional. Para acessar a Área de trabalho remota, use o seguinte caminho:

Iniciar > Todos os programas > Acessórios > Comunicações > Conexão de área de trabalho remota

Configurações de desempenho
Para melhorar o desempenho do sistema operacional, você pode alterar algumas configurações do seu computador, como as configurações de memória virtual, que aparecem na Figura 4. Para alterar a configuração de memória virtual, use o seguinte caminho:

Iniciar > Painel de controle > Sistema > Avançado > Desempenho > botão Configurações
