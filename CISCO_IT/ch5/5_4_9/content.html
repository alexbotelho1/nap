5.4 Instalação de um sistema operacional  
  5.4.9 Descrição das estruturas de diretório  

Atributos e extensões de arquivo
No Windows, os arquivos são organizados em uma estrutura de diretórios. O nível raiz da partição do Windows é em geral a unidade C:\. Em seguida, há um conjunto inicial de diretórios padronizados, chamados pastas, para o sistema operacional, aplicativos, informações de configuração e arquivos de dados. Após a instalação inicial, os usuários podem instalar a maioria dos aplicativos e dados em qualquer diretório de sua escolha.

Os arquivos na estrutura de diretórios obedecem à convenção de nomeação do Windows.

  • Podem ser usados até 255 caracteres.
  • Caracteres como ponto (.) ou barra (\ /) não são permitidos.
  • Uma extensão de três ou quatro letras é adicionada ao nome do arquivo para identificar o tipo de arquivo.
  • Os nomes de arquivo não diferenciam maiúsculas de minúsculas.

As seguintes extensões de nome de arquivo são geralmente utilizadas:

  • .doc – Microsoft Word
  • .txt – somente texto ASCII
  • .jpg – formato de elemento gráfico
  • .ppt – Microsoft PowerPoint
  • .zip – formato de compactação

A estrutura de diretórios mantém um conjunto de atributos para cada arquivo que controla como o arquivo pode ser exibido ou alterado. Estes são os atributos de rede mais comuns:

  • R – o arquivo é somente leitura.
  • A – o arquivo será armazenado no próximo backup do disco.
  • S – o arquivo é marcado como arquivo de sistema e um aviso é emitido se ocorrer alguma tentativa de excluí-lo ou modificá-lo.
  • H – o arquivo é oculto na exibição de diretórios.

Para exibir os nomes de arquivo, as extensões e os atributos, abra uma janela do DOS e use o comando ATTRIB, como mostra a Figura 1. Use o seguinte caminho:

Iniciar > Executar > cmd

Navegue até a pasta que contém o arquivo de seu interesse. Digite ATTRIB seguido do nome do arquivo. Use um curinga como *.* para exibir muitos arquivos de uma vez. Os atributos de cada arquivo aparecerão na coluna esquerda da tela. Para encontrar informações sobre o comando ATTRIB no prompt de comando, digite:

ATTRIB/?

Para acessar o equivalente do Windows para o atributo ATTRIB, clique com o botão direito do mouse em um arquivo do Windows Explorer e escolha Propriedades.

OBSERVAÇÃO: Para ver as propriedades de um arquivo no Windows Explorer, primeiro defina o Windows Explorer para "Mostrar arquivos ocultos". Use este caminho:

Clique com o botão direito do mouse em Iniciar > Pesquisar > Ferramentas > Opções de pasta > Exibir

NTFS e FAT32
O Windows XP e o Windows 2000 usam sistemas de arquivos FAT32 e NTFS. Segurança é uma das diferenças mais importantes entre esses sistemas de arquivos. O NTFS aceita mais arquivos e arquivos maiores do que o FAT32 e oferece recursos de segurança mais flexíveis para arquivos e pastas. As Figuras 2 e 3 mostram as propriedades de permissão de arquivos para FAT32 e NTFS.

As partições podem ser convertidas de FAT32 em NTFS com o utilitário CONVERT.EXE. Esse procedimento garantirá as vantagens extras de segurança do NTFS. Para reverter uma partição NTFS em uma partição FAT32, reformate a partição e restaure os dados de um backup.

CUIDADO: Antes de converter o sistema de arquivos, lembre-se de fazer backup dos dados.
