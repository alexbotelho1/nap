5.8 Resumo  

Este capítulo apresentou sistemas operacionais de computador. Como técnico, você deve estar habilitado a instalar, configurar e solucionar problemas de um sistema operacional. É importante lembrar os seguintes conceitos deste capítulo:

  • Existem vários sistemas operacionais diferentes disponíveis e você deve levar em consideração as necessidades do cliente e o ambiente ao escolher um sistema operacional.
  • As etapas principais ao configurar o computador de um cliente incluem preparar o disco rígido, instalar um sistema operacional, criar contas de usuário e configurar opções de instalação.
  • Uma GUI mostra ícones de todos os arquivos, pastas e aplicativos no computador. Um dispositivo apontador, como um mouse, é usado para navegar em uma área de trabalho de GUI.
  • Você deve estabelecer uma estratégia de backup que permita a recuperação de dados. Backups normais, de cópia, diferenciais, incrementais e diários são todos ferramentas opcionais de backup disponíveis nos sistemas operacionais Windows.
  • As técnicas de manutenção preventiva ajudam a garantir uma operação ideal do sistema operacional.
  • Algumas das ferramentas disponíveis para solucionar problemas do sistema operacional incluem o menu de Opções avançadas do Windows, os logs de eventos, o gerenciador de dispositivos e os arquivos do sistema.
