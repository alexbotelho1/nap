5.4 Instalação de um sistema operacional  
  5.4.5 Conclusão da instalação  

Depois que a instalação do Windows copia todos os arquivos necessários do sistema operacional para a unidade de disco rígido, o computador é reinicializado e solicita o seu login pela primeira vez.

Você deve registrar o Windows XP. Como mostra a Figura 1, você também deve concluir a verificação que garante o uso legal da cópia do sistema operacional. Dessa forma, você poderá fazer download de patches e service packs. A realização desta etapa requer uma conexão com a Internet.

Dependendo da idade da mídia no momento da instalação, pode haver atualizações a serem instaladas. De acordo com a Figura 2, você pode usar o Gerenciador de atualizações da Microsoft no menu Iniciar para procurar novos softwares e fazer o seguinte:

  • Instalar todos os service packs
  • Instalar todos os patches

Iniciar > Todos os programas > Acessórios > Ferramentas do sistema > Windows Update

Verifique também se os hardwares foram instalados corretamente. Como indicado na Figura 3, você pode usar o Gerenciador de dispositivos para localizar problemas e instalar os drivers corretos e atualizados usando o seguinte caminho:

Iniciar > Painel de controle > Sistema > Hardware > Gerenciador de dispositivos

No Gerenciador de dispositivos, os ícones de aviso são representados por um ponto de exclamação amarelo ou um “X” vermelho. O ponto de exclamação amarelo representa um problema com o dispositivo. Para ver a descrição do problema, clique com o botão direito do mouse no dispositivo e selecione Propriedades. O “X” vermelho representa um dispositivo que foi desativado. Para ativar o dispositivo, clique com o botão direito do mouse no dispositivo desativado e selecione Ativar. Para abrir uma categoria que ainda não tenha sido expandida, clique no sinal de mais (+).

OBSERVAÇÃO: Quando o Windows detecta um erro do sistema, o recurso de relatórios do Windows exibe uma caixa de diálogo. Se você optar por enviar o relatório, o recurso Microsoft WER (Windows Error Reporting, relatórios de erro do Windows) coletará informações sobre o aplicativo e o módulo envolvido no erro e enviará as informações para a Microsoft.
