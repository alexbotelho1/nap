5.6 Identificar e aplicar as técnicas comuns de manutenção preventiva para sistemas operacionais  
  5.6.2 Agendamento de uma tarefa  

A manutenção preventiva consiste em limpar, inspecionar e fazer pequenos reparos, além de utilizar ferramentas de aplicativos já presentes no sistema operacional ou que podem ser carregadas no disco rígido do usuário. A maioria dos aplicativos de manutenção preventiva pode ser configurada para ser executada automaticamente de acordo com uma programação.

O Windows possui os seguintes utilitários que iniciarão tarefas quando você programá-las:

  • O comando AT do DOS inicia tarefas em um determinado horário usando a interface de linha de comando.
  • O Agendador de Tarefas do Windows inicia tarefas em um determinado horário usando uma interface gráfica do usuário.

Informações sobre o comando AT estão disponíveis neste caminho:

Iniciar > Executar > cmd

Em seguida, digite AT /? na linha de comando.

Acesse o Agendador de Tarefas do Windows seguindo este caminho:

Iniciar > Todos os programas > Acessórios > Ferramentas do Sistema > Tarefas agendadas

Essas duas ferramentas permitem aos usuários configurar comandos para serem executados em um determinado momento apenas uma vez ou para serem repetidos em determinados dias ou horários. O Agendador de Tarefas do Windows, exibido na Figura 1, é mais fácil de ser aprendido e usado do que o comando AT, especialmente quando se trata de tarefas recorrentes e de excluir tarefas já programadas.

Utilitários do sistema
Existem vários utilitários incluídos no DOS e no Windows que ajudam a manter a integridade do sistema. Há dois utilitários que são ferramentas úteis para uma manutenção preventiva:

  • ScanDisk ou CHKDSK – O ScanDisk (Windows 2000) e o CHKDSK (Windows XP) verificam a integridade de arquivos e pastas e examinam a superfície do disco rígido em busca de erros físicos. Use-os pelo menos uma vez por mês e também sempre que uma perda repentina de energia causar o desligamento do sistema.
  • Defrag – À medida que os arquivos aumentam de tamanho, alguns dados são gravados no próximo espaço disponível do disco. Com o tempo, os dados ficam fragmentados ou espalhados por todo o disco rígido. Leva tempo para procurar cada seção dos dados. O Defrag reúne os dados não contíguos em um lugar, fazendo com que os arquivos sejam executados mais rapidamente.

Você pode acessar esses dois utilitários usando este caminho:

Iniciar > Todos os programas > Acessórios > Ferramentas do Sistema > Desfragmentador de disco

Atualizações automáticas
Se toda tarefa de manutenção tivesse que ser programada a cada vez que ela fosse executada, consertar computadores seria muito mais difícil do que é hoje. Felizmente, ferramentas como o Assistente de tarefa agendada permitem que várias funções sejam automatizadas. Mas como você pode automatizar a atualização do software que não foi gravado?

Os sistemas operacionais e os aplicativos estão sendo constantemente atualizados por razões de segurança e para uma maior funcionalidade. É importante que a Microsoft e outras empresas forneçam um serviço de atualização, conforme exibido na Figura 2. O serviço de atualização pode verificar o sistema em busca de atualizações necessárias e, em seguida, recomendar o que deve ser baixado e instalado. O serviço de atualização pode baixar e instalar atualizações assim que elas estiverem disponíveis ou conforme necessário e instalá-las quando o computador for reinicializado da próxima vez. O Assistente de atualização da Microsoft está disponível neste caminho:

Iniciar > Painel de controle > Sistema > Atualizações Automáticas

A maioria dos softwares antivírus contém seu próprio recurso de atualização. Ele pode atualizar tanto seu software de aplicativo como seus arquivos do banco de dados automaticamente. Essa função permite que ele forneça uma proteção imediata à medida que novas ameaças são reveladas.

Ponto de restauração
Uma atualização pode às vezes causar sérios problemas. Talvez exista no sistema um programa mais antigo não compatível com o sistema operacional atual. Uma atualização automática pode instalar códigos que funcionarão para a maioria dos usuários, mas não funcionarão com o seu sistema.

O Ponto de restauração do Windows, exibido na Figura 3, é a solução para esse problema. O Windows XP pode criar uma imagem das configurações atuais do computador, chamada de ponto de restauração. Então, se o computador travar ou se uma atualização causar problemas no sistema, o computador poderá ser revertido para uma configuração anterior.

Um técnico deve sempre criar um ponto de restauração antes de atualizar ou substituir o sistema operacional. Os pontos de restauração também devem ser criados nas seguintes situações:

  • Quando um aplicativo é instalado
  • Quando um driver é instalado

OBSERVAÇÃO: Um ponto de restauração faz o backup de drivers, arquivos do sistema e configurações do Registro, mas não de dados de aplicativos.

Para restaurar ou criar um ponto de restauração, use o seguinte caminho:

Iniciar > Todos os programas > Acessórios > Ferramentas do Sistema > Restauração do sistema

ERD e ASR
O Windows 2000 proporciona a criação de um ERD (Emergency Repair Disk, disco de reparação de emergência) que salva arquivos de inicialização essenciais e informações de configuração necessárias para solucionar problemas no Windows. O Windows XP oferece os mesmos recursos com o assistente para ASR (Automated System Recovery, recuperação automatizada do sistema). Embora o ERD e o ASR sejam ferramentas poderosas de solução de problemas, eles nunca devem substituir um bom backup.

Um CD de recuperação contém os arquivos essenciais usados para reparar o sistema após um problema sério, como pane no disco rígido. O CD de recuperação pode conter a versão original do Windows, drivers de hardware e software de aplicativo. Quando o CD de recuperação for usado, o computador será restaurado para a configuração padrão original.
