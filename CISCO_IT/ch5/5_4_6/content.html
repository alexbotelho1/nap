5.4 Instalação de um sistema operacional  
  5.4.6 Descrição das opções de instalação personalizadas  

A instalação de um sistema operacional em um único computador é demorada. Imagine o tempo para instalar sistemas operacionais em vários computadores, um de cada vez, em uma grande empresa. Para simplificar essa atividade, você pode usar a ferramenta Sysprep (Microsoft System Preparation) para instalar e configurar o mesmo sistema operacional em vários computadores. O Sysprep prepara o sistema operacional que será usado em computadores com diversas configurações de hardware. Com o Sysprep e um aplicativo de clonagem de disco, os técnicos podem rapidamente instalar um sistema operacional, concluir as últimas etapas de configuração do sistema operacional e instalar aplicativos.

Clonagem de disco
A clonagem de disco cria uma imagem da unidade de disco rígido em um computador. Siga estas etapas para a clonagem de disco:

  1. Crie uma instalação mestra em um computador. Essa instalação mestra inclui o sistema operacional, aplicativos de software e definições de configuração que serão usados por outros computadores da empresa.
  2. Execute o Sysprep.
  3. Crie uma imagem de disco do computador configurado usando um programa de clonagem de disco de terceiros.
  4. Copie a imagem de disco para um servidor. Quando o computador de destino é inicializado, uma versão reduzida do programa de instalação do Windows é executada. O programa de instalação criará um novo SID (System Security Identifier, identificador de segurança do sistema), instalará drivers para hardware e criará contas de usuário, além de definir as configurações de rede para concluir a instalação do sistema operacional.
