  5.5.4 Instalação, navegação e desinstalação de um aplicativo  

Como técnico, você será responsável por adicionar e remover software nos computadores dos clientes. Muitos aplicativos usam um processo de instalação automática quando o CD do aplicativo é inserido na unidade óptica. O processo de instalação atualiza o utilitário Adicionar ou remover programas. O usuário será solicitado a clicar no Assistente de instalação para fornecer informações.

Applet Adicionar ou remover programas
A Microsoft recomenda usar o utilitário Adicionar ou remover programas, como mostra a Figura 1, para a instalação ou remoção de aplicativos. Quando usado para instalar um aplicativo, o utilitário Adicionar ou remover programas rastreia os arquivos de instalação para que o aplicativo possa ser desinstalado por completo, caso desejado. Para abrir o applet Adicionar ou remover programas, use o seguinte caminho:

Iniciar > Painel de controle > Adicionar ou remover programas

Adicionar aplicativos
Se um programa ou um aplicativo não é instalado automaticamente quando o CD é inserido, você pode usar o applet Adicionar ou remover programas para instalar o aplicativo, como mostra a Figura 2. Clique no botão Adicionar novos programas e selecione o local onde o aplicativo está localizado. O Windows instalará o aplicativo para você.

Depois de instalado, o aplicativo poderá ser iniciado pelo menu Iniciar ou por um ícone de atalho instalado na área de trabalho. Verifique se o aplicativo está funcionando corretamente. Se houver problemas, faça o reparo ou desinstale o aplicativo. Alguns aplicativos, como o Microsoft Office, oferecem uma opção de reparo no processo de instalação. Você pode usar essa função para tentar corrigir um programa que não esteja funcionamento adequadamente.

Desinstalar aplicativos
Se algum aplicativo não for desinstalado corretamente, talvez fiquem arquivos na unidade de disco rígido e configurações desnecessárias no Registro. Embora essa situação não cause problemas, ela consome espaço disponível no disco rígido e recursos do sistema, e prejudica a velocidade com que o Registro é lido. A Figura 3 mostra o applet Adicionar ou remover programas para desinstalar programas que não são mais necessários. O assistente o orientará no processo de remoção do software e excluirá todos os arquivos que foram instalados.
