5.3 Determinação do sistema operacional com base nas necessidades do cliente  

Para selecionar o sistema operacional apropriado para as necessidades do seu cliente, você precisa saber como ele pretende usar o computador. O sistema operacional recomendado deve ser compatível com os aplicativos que serão usados e com todo o hardware instalado no computador. Se o computador estiver conectado a uma rede, o novo sistema operacional também deverá ser compatível com outros sistemas operacionais da rede.

Após a conclusão desta seção, você estará apto a:

  • Identificação de aplicativos e ambientes que sejam compatíveis com o sistema operacional.
  • Determinação dos requisitos mínimos de hardware e a compatibilidade com a plataforma do sistema operacional.
