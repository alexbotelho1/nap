5.4 Instalação de um sistema operacional  
  5.4.3 Instalação do sistema operacional usando as configurações padrão  

O assistente de instalação do Windows XP lhe oferece a opção de instalar usando configurações típicas (padrão) ou personalizadas. As configurações típicas aumentam a probabilidade de uma instalação bem-sucedida. Contudo, o usuário ainda precisará fornecer as seguintes informações durante a configuração:

  • Padrões e formatos que definem moeda e numerais
  • Idioma do texto
  • Nome do usuário e da empresa
  • Chave do produto
  • Nome do computador
  • Senha do administrador
  • Configurações de data e hora
  • Configurações de rede
  • Informações sobre domínio ou grupo de trabalho

Quando o computador inicializa com o CD de instalação do Windows, a instalação do Windows XP começa com três opções:

  • Configurar o XP – para configurar e instalar o sistema operacional XP, pressione ENTER.
  • Reparar o XP – para reparar uma instalação, pressione R para abrir o Console de recuperação.
  • Sair – para sair sem instalar o Windows XP, pressione F3.

Para esta seção, selecione a opção de configurar o XP.
