  5.5.5 Descrição da atualização de um sistema operacional  

Pode haver ocasiões em que seja necessário atualizar o sistema operacional. Antes de atualizar o sistema operacional, verifique os requisitos mínimos do novo sistema para garantir que o computador atenda às especificações mínimas necessárias. Verifique a lista HCL para certificar-se de que o hardware seja compatível com o novo sistema operacional. Faça backup de todos os dados antes de atualizar o sistema operacional para prevenir-se se houver algum problema com a instalação.

O processo de atualização do sistema de computador do Windows 2000 para o Windows XP é mais rápido do que uma nova instalação do Windows XP. O utilitário de instalação do Windows XP substitui os arquivos existentes do Windows 2000 pelos do Windows XP durante o processo de atualização. Contudo, os aplicativos e as configurações existentes serão salvos.

Atualizando o sistema operacional

  1. Insira o CD do Windows XP na unidade de CD-ROM para inicar o processo de atualização. Selecione Iniciar > Executar.
  2. Na caixa Executar, onde D é a letra da unidade de CD-ROM, digite D:\i386\winnt32 e pressione Enter. A janela de boas vindas do Assistente de instalação do Windows XP é exibida.
  3. Escolha a opção de Atualizar para o Windows XP e clique em Avançar. A página do contrato de licença é exibida.
  4. Leia o contrato de licença e clique no botão para aceitá-lo.
  5. Clique em Avançar. A página da atualização para o sistema de arquivos NTFS do Windows XP é exibida.
  6. Siga os prompts e conclua a atualização. Quando a instalação estiver concluída, o computador será reiniciado.

OBSERVAÇÃO: O Assistente de instalação do Windows XP pode iniciar automaticamente quando o CD for inserido na unidade óptica.
