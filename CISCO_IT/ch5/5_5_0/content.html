5.5 Navegação na GUI (Windows)  

O sistema operacional oferece uma interface de usuário que lhe permite interagir com o computador. Existem dois métodos para se navegar no sistema de arquivos e executar aplicativos em um sistema operacional.

  • A GUI (Graphical User Interface, interface gráfica do usuário) fornece representações gráficas (ícones) de todos os arquivos, pastas e programas do computador. Você manipula esses ícones com um cursor que é controlado com um mouse ou dispositivo semelhante. O cursor lhe permite arrastar e soltar ícones, e executar programas com um clique.
  • A CLI (Command Line Interface, interface de linha de comando) se baseia em texto. Você precisa digitar comandos para manipular arquivos e executar programas.

Após a conclusão desta seção, você estará apto a:

  • Manipulação de itens na área de trabalho.
  • Exploração de applets do Painel de controle.
  • Exploração de ferramentas administrativas.
  • Instalação, navegação e desinstalação de um aplicativo.
  • Descrição da atualização de sistemas operacionais.
