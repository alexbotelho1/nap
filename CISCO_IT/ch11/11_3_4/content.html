11.3 Descrever situações que exigem a substituição de componentes do computador  
  11.3.4 Selecionar RAM  

Pode ser necessária uma nova RAM quando um aplicativo trava ou o computador exibe mensagens de erro freqüentemente. Para determinar se o problema é a RAM, substitua o módulo de RAM antigo, conforme mostrado na Figura 1. Reinicie o computador para verificar se o aplicativo é executado corretamente.

OBSERVAÇÃO:Para fechar um aplicativo bloqueado, pressione Crtl-Alt-Delete para abrir o Gerenciador de tarefas. Selecione o aplicativo na janela. Clique em Finalizar tarefa para fechá-lo, conforme mostrado na Figura 2.

Ao selecionar a nova RAM, você deve certificar-se de que ela é compatível com a placa-mãe atual. Ela também deve ser do mesmo tipo de RAM já instalada no computador. A velocidade da nova RAM deve ser igual ou maior que a da RAM existente. Pode ser útil levar o módulo de memória original quando você for comprar a RAM de substituição.
