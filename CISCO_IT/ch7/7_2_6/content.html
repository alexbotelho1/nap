7.2 Descrever os processos de instalação e configuração de impressoras  
  7.2.6 Descrever como imprimir uma página de teste  

Após instalar uma impressora, você deverá imprimir uma página de teste para verificar se a impressora está funcionando corretamente. A página de teste confirma se o software do driver foi instalado e está funcionando corretamente e se a impressora e o computador estão se comunicando.

Imprimir uma página de teste
Para imprimir uma página de teste manualmente, use o seguinte caminho:
Iniciar > Impressoras e aparelhos de fax para exibir o menu Impressoras e aparelhos de fax.

Clique com o botão direito do mouse na impressora desejada e siga este caminho:
Propriedades > Guia Geral > Imprimir página de teste

Uma caixa de diálogo será aberta, perguntando se a página foi impressa corretamente. Se a página não foi impressa, os arquivos da ajuda incorporados o ajudarão a solucionar o problema.

Imprimir a partir de um aplicativo
Você também pode testar uma impressora imprimindo uma página de teste a partir de um aplicativo como o Bloco de notas ou o WordPad. Para acessar o Bloco de notas, use o seguinte caminho:
Iniciar > Programas > Acessórios > Bloco de notas

Um documento em branco será aberto. Digite algum texto no documento. Imprima-o usando o seguinte caminho:
Arquivo > Imprimir

Testar uma impressora
Você também pode imprimir a partir da linha de comando para testar a impressora. A impressão a partir da linha de comando é limitada apenas a arquivos ASCII, como arquivos .txt e .bat. Para enviar um arquivo à impressora a partir da linha de comando, use este caminho:
Iniciar > Executar

A caixa Executar deve ser exibida. Digite cmd na caixa Executar e clique em OK.

No prompt da linha de comando, digite o seguinte comando:
Imprimir oarquivo.txt

Testar a impressora a partir do painel da impressora
A maioria das impressoras possui um painel frontal com controles para permitir que você gere páginas de teste. Esse método de impressão permite que você verifique o funcionamento da impressora separadamente da rede ou do computador. Consulte o site do fabricante da impressora ou a documentação para aprender como imprimir uma página de teste a partir do painel frontal da impressora.
