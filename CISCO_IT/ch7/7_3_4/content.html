  7.3.4 Descrever os scanners portáteis  

Um scanner portátil é pequeno e cabe na sua mão. É difícil digitalizar corretamente uma imagem usando um scanner portátil. Para digitalizar um item, passe cuidadosamente a cabeça do scanner pelo item que deseja digitalizar. Como ocorre com o scanner de mesa, as imagens digitais são feitas das imagens coletadas pelo scanner portátil.

Quando você desejar digitalizar um item maior que a cabeça do scanner portátil, deverá fazer mais de uma passagem para capturar a imagem inteira. Pode ser difícil recriar a imagem original digitalmente quando ela for digitalizada com mais de uma passagem. As imagens devem ser reunidas para formar uma única imagem do item digitalizado.
