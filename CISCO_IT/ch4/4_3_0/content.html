4.3 Resumo  

Este capítulo abordou os conceitos de manutenção preventiva e de solução de problemas.

  • A manutenção preventiva regular reduz problemas de hardware e software.
  • Antes de iniciar qualquer reparo, faça backup dos dados do computador.
  • O processo de solução de problemas é uma diretriz para ajudá-lo a solucionar problemas do computador com eficiência.
  • Documente todas as suas tentativas, mesmo as que falharem. A documentação criada será um recurso útil para você e outros técnicos.
