safe mode (modo de segurança)

Opção de inicialização do sistema que carrega somente os dispositivos básicos que o Windows precisa executar. Esse processo é usado para a solução de problemas.

satellite communication (comunicação por satélite)

Utilização de satélites orbitais para transmitir dados entre múltiplas estações terrestres. A comunicação por satélite oferece alta largura de banda e capacidade de transmissão sob um custo que não está relacionado à distância entre as estações terrestres. Devido à altitude do satélite, esse tipo de comunicação pode estar sujeito a longos atrasos de propagação.

SCANDISK

Utilitário do Windows usado para examinar todos os arquivos em uma unidade.

security key fob (chave de segurança)

Pequeno sistema de rádio que se comunica com o computador por um breve intervalo. O computador deve receber o sinal da chave de segurança antes de aceitar os dados de nome de usuário e senha.

security keyhole (ponto da chave de segurança)

Ponto rígido no gabinete usado para conectar um cabo de segurança.

segment (segmento)

Parte de uma rede de computador na qual o dispositivo se comunica usando a camada física do modelo OSI. Hubs e repetidores são estendidos e tornam-se parte de um segmento de rede, enquanto os switches e roteadores definem e separam os segmentos de rede.

semicondutor

Material usado para criar chips de computador que podem ser usados como condutor ou como isolante, dependendo dos sinais de controle aplicados a ele. Os materiais mais comuns de um semicondutor são o silicone e o germânio, e estes materiais contêm outros materiais adicionados a eles que aumentam a condutividade.

serial cable (cabo serial)

Cabo externo que conecta a porta serial do computador a um dispositivo periférico.

serial data transfer (transferência de dados serial)

Movimento de bits simples de informação em um único ciclo.

serial port (porta serial)

Soquete usado para conectar um dispositivo, como mouse ou trackball.

serial transmission (transmissão serial)

Método de transmissão de dados no qual os bits de um caractere de dados são transmitidos seqüencialmente através de um único canal.

server (servidor)

Repositório para arquivos, ou outros recursos, que pode ser acessado e compartilhado pela rede por muitos usuários.

SLA (service level agreement, contrato de nível de serviço)

Contrato que define as expectativas entre uma organização e o fornecedor do serviço para fornecer um nível de suporte acordado.

session layer (camada de sessão)

Camada 5 do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos). Essa camada estabelece, gerencia e encerra sessões entre aplicações e gerencia troca de dados entre entidades de camadas de apresentação. O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

STP (shielded twisted pair, par trançado blindado)

Meio composto por dois pares de fios usados principalmente em redes token ring. O cabeamento do STP possui uma camada de isolamento blindado para reduzir a interferência eletromagnética (EMI). Compare com UTP.

shortcut (atalho)

Combinação no teclado que ativa um comando.

SMTP (Simple Mail Transfer Protocol, protocolo de transferência de correio simples)

Protocolo de e-mail usado por servidores para enviar mensagens de texto ASCII. Quando ampliado pelo protocolo MIME, o SMTP pode enviar e-mails com imagens e documentos. Algumas vezes, o SMTP é usado por clientes de e-mail para recuperar mensagens a partir de um servidor de e-mails. Entretanto, devido à capacidade limitada para as mensagens em fila ao fim de uma recepção, outros protocolos, como POP ou IMAP, são utilizados para receber e-mails.

simplex

Capacidade de transmissão de dados em apenas uma direção entre a estação de envio e uma estação de recebimento.

single-mode (modo simples)

Fibra óptica com núcleo muito fino. Usa alta energia a laser como fonte de luz e pode transmitir sinais a distâncias mais longas que o cabo de fibra óptica multimodo.

site survey (pesquisa de local)

Inspeção física do local que ajudará a determinar uma topologia de rede básica.

SCSI (Small Computer System Interface, interface de sistema de computadores de pequeno porte)

Padrão de interface paralela que suporta vários dispositivos no mesmo cabo e atinge taxas de transmissão de dados mais rápidas que os barramentos padrão.

Small Computer System Interface (cabo da SCSI (Small Computer System Interface, interface de sistema de computadores de pequeno porte) )

Cabo externo ou interno que conecta o controlador SCSI às portas SCSI de vários dispositivos internos e externos.

smart card (cartão inteligente)

Dispositivo do tamanho de um cartão de crédito que inclui processador e memória usado para armazenar informações e autenticar os usuários da rede. Os cartões inteligentes oferecem identificação dupla, pois o usuário deve ter tanto o cartão quanto a senha para acessar a rede.

software firewall (firewall de software)

Aplicativo em um computador que inspeciona e filtra os pacotes de dados.

solenoid (solenóide)

Bobina de fios que forma eletromagnéticos que acionam os pinos na impressora matricial.

solid-ink printers (impressoras de tinta sólida)

Impressora que usa bastões de tinta sólida em vez de toner ou cartuchos de tinta. Impressoras de tinta sólida produzem imagens de alta qualidade. Os bastões de tinta não são tóxicos e podem ser manipulados com segurança.

sound card (placa de som)

Placa de circuito integrado que aprimora os recursos de áudio do computador.

southbridge

Chip que implementa os recursos mais lentos da placa-mãe. É conectado à CPU pelo chip northbridge. Também é conhecido como ICH (Controlling Station, estação de controle) de E/S (Input/Output, entrada/saída).

spam

E-mail não solicitado.

speaker (alto-falante)

Dispositivo de saída de áudio.

spike (pico)

Aumento repentino na voltagem normalmente causado por relâmpagos.

spoof

Para obter acesso aos recursos dos dispositivos tentando passar por um computador confiável.

spooling

Processo de carregamento de documentos para um buffer (normalmente uma área no disco rígido) até que a impressora esteja pronta para imprimir os documentos.

spyware

Malware que monitora a atividade em um computador. O spyware envia as informações para a organização responsável pelo lançamento do spyware.

SPS (standby power supply, fonte de energia de reserva)

Reserva de energia que é ativada quando os níveis de voltagem caem abaixo do normal.

standoff (separador)

Obstáculo usado para separar fisicamente as peças.

star topology (topologia estrela)

Topologia de uma rede local na qual os pontos finais de uma rede são conectados a um switch central comum através de vínculos ponto-a-ponto. Uma topologia em anel que é organizada como uma estrela e implementa uma estrela de loop fechado unidirecional em vez de vínculos ponto-a-ponto.

SRAM (RAM estática)

Tipo de memória RAM que retém o conteúdo enquanto o dispositivo estiver ligado. A RAM estática não requer recuperação constante como a RAM dinâmica.

SRAM (static random access memory, memória de acesso aleatório estática)

Memória que retém dados enquanto houver voltagem aplicada. Usado principalmente como memória cache para a CPU.

stylus (caneta)

Utensílio de escrita. Geralmente usado como um método de entrada para as telas sensíveis ao toque de PDAs e tablets gráficos.

subnet mask (máscara de sub-rede)

Segundo grupo de números usado na configuração de um endereço IP em um dispositivo. A máscara de sub-rede é usada por dispositivos finais para determinar a parte da rede de um endereço IP.

subnetting (sub-rede)

Divisão lógica de uma rede. Ela fornece os meios para dividir uma rede e a máscara de sub-rede especifica como ela é subdividida.

surge (surto)

Toda voltagem que aumenta mais de 110% da voltagem normal transportada por uma linha de energia.

surge protector (protetor contra surto de voltagem)

Protetor que regula a voltagem que chega a um dispositivo.

surge suppressor (protetor anti-surto)

Dispositivo que verifica se a voltagem que chega a outro dispositivo está abaixo de um determinado nível.

S-video port (porta S-video)

Miniconector DIN de quatro pinos usado para emitir os sinais de vídeo a um dispositivo compatível. O S-video separa o brilho e a cor de um sinal de vídeo.

switch

1) Operação adicionada a um comando do DOS para modificar o resultado desse comando. 2) Um dispositivo de rede de Camada 2 também conhecido como ponte multiporta.

SDSL (Symmetric DSL, linha digital de assinantes simétrica)

Versão de um serviço DSL (Digital Subscriber Loop, loop digital de assinantes) que oferece a mesma velocidade para uploads e downloads.

Symmetric encryption (criptografia simétrica)

Criptografia que requer ambas as partes de uma conversa criptografada para usar uma chave de criptografia capaz de codificar e decodificar os dados. O remetente e o receptor devem usar a mesma chave ao mesmo tempo.

SYN flood (inundação SYN)

Abre aleatoriamente as portas TCP, bloqueando temporariamente o equipamento de rede ou os recursos do computador devido a uma grande quantidade de solicitações falsas, fazendo com que as sessões sejam negadas aos outros usuários. Veja também DoS (Denial of Service, ataques de negação de serviços).

System Restore (Restauração do sistema)

Serviço do Windows XP que é executado em segundo plano e que permite ao usuário restaurar o sistema operacional para um período de tempo predefinido.
