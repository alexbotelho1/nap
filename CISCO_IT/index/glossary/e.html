eject button (botão ejetar)

Libera um objeto, como o botão em uma unidade de disquete.

EIA (Electronic Industries Association, associação das indústrias eletrônicas)

Grupo que especifica os padrões de transmissão elétrica. A EIA e o TIA desenvolveram vários padrões de comunicação bastante conhecidos, incluindo o EIA/TIA-232 e o EIA/TIA-449.

e-mail (correio eletrônico)

Habilidade que permite aos usuários se comunicarem por uma rede de computador. Troca de mensagens armazenadas no computador por meio de uma comunicação via rede.

electrophotographic drum (tambor eletrofotográfico)

Parte central da impressora a laser que usa o toner para imprimir no papel.

ESD (electrostatic discharge, descarga eletrostática)

Descarga de eletricidade estática de um condutor para outro de potências diferentes.

encryption (criptografia)

Recurso de segurança que aplica uma codificação em um arquivo de forma que apenas os usuários autorizados possam visualizar o arquivo.

EFS (Encryption File System, criptografia de sistema de arquivos)

Sistema de arquivos para criptografia específico da Microsoft.

EIDE (Enhanced Integrated Drive Electronics)

Versão aprimorada da interface IDE padrão que conecta discos rígidos, unidades de CD-ROM e unidades de fita a um PC.

Ethernet

Especificação de LAN de banda base criada pela Xerox Corporation e desenvolvida em conjunto pela Xerox, Intel e pela Digital Equipment Corporation. As redes Ethernet usam o CSMA/CD e são executadas por uma variedade de tipos de cabos a 10 Mbps ou mais. A Ethernet é semelhante com a série de padrões IEEE 802.3.

Ethernet port (porta Ethernet)

Soquete RJ-45 usado para conectar um computador a uma rede de área local (LAN) por cabo.

event (evento)

Mensagem da rede que indica irregularidades operacionais nos elementos físicos de uma rede ou uma resposta para a ocorrência de uma tarefa significativa, geralmente a conclusão de uma solicitação de informações.

Event Viewer (Visualizar eventos)

Aplicativo que monitora eventos do sistema, eventos do aplicativo e eventos de segurança.

exhaust vent (aberturas de exaustão)

Saída que libera ar quente do interior de um dispositivo ou sala.

expansion card modem (modem da placa de expansão)

Modem inserido no slot de expansão da placa-mãe (ISA ou PCI). Também chamado de modem interno.

expansion slot (slot de expansão)

Abertura no computador que permite a inserção de uma placa de PC para adicionar recursos ao computador.

ExpressCard

Placa padrão de expansão de laptop e de alto-processamento desenvolvida pela PCMCIA. O slot de expansão ExpressCard usa a PCI Express (x1) incorporada e/ou o barramento USB de um laptop. As ExpressCards possuem um conector de 26 pinos e são intercambiáveis.

extended memory (memória estendida)

Memória superior a 1 MB.

extended partition (partição estendida)

Segunda partição no disco rígido.

extended-star topology (topologia estrela estendida)

Topologia estrela expandida para incluir dispositivos de rede adicionais.

external hard drive (disco rígido externo)

Dispositivo que é conectado ao computador para fornecer armazenamento de dados adicional.

external modem (modem externo)

Modem que é conectado à porta serial (COM1 ou COM2) da maioria dos computadores. Um modem externo, como um cable modem, normalmente é usado em conexões de alta velocidade.
