AC power connector (conector de energia CA)

Soquete usado para conectar o adaptador de energia CA a um computador ou base multifuncional.

AC power cord (cabo de energia CA)

Cabo que transfere eletricidade da fonte de energia CA para a fonte de energia do computador.

AGP (Accelerated Graphics Port, porta de gráficos acelerada)

Tecnologia de barramento de 32 bits, de alta velocidade desenvolvida para oferecer suporte à aceleração de gráficos em 3D no computador.

ACL (Access Control List, lista de controle de acesso)

Lista gerenciada por um administrador de rede que registra os detalhes sobre o que um usuário pode acessar e o tipo de acesso concedido.

access point (ponto de acesso)

Dispositivo que conecta dispositivos sem fio para formar uma rede sem fio. Um ponto de acesso geralmente é conectado a uma rede cabeada e pode restabelecer os dados entre dispositivos com e sem fio. As distâncias de conectividade podem ser de vários pés ou metros a várias milhas ou quilômetros.

active partition (partição ativa)

Partição em uma unidade de disco rígido que é definida como a partição inicializável e normalmente contém o sistema operacional a ser usado no computador. No computador, apenas uma partição pode ser definida como partição inicializável ou ativa na unidade de disco rígido.

ActiveX

Applet ou pequeno programa criado pela Microsoft para controlar a interatividade nas páginas da Web que devem ser baixadas para obter acesso a todas as funções.

adapter card (placa do adaptador)

Placa de expansão que aumenta o número de controladores e portas disponíveis em um computador.

ARP (Address Resolution Protocol, protocolo de resolução de endereços)

Descobre o endereço local (endereço MAC) de uma estação na rede quando o endereço IP é desconhecido. Estações finais e roteadores usam o ARP para descobrir endereços locais:
-a Switch usado com o comando ARP que exibe o cache.
-d Switch usado com o comando ARP que exclui uma entrada do cache ARP.
-s Switch usado com o comando ARP que adiciona um mapeamento permanente de endereço IP para MAC.

administrator(administrador)

Pessoa que consulta o Registro de Usuário para analisar o status de assinante individual e para coletar dados.

ACPI (Advanced Configuration and Power Interface, interface de energia e configuração avançada)

Interface que permite ao sistema operacional controlar o gerenciamento de energia. Substitui o APM (Advanced Power Management, gerenciamento avançado de energia).

APM (Advanced Power Management, gerenciamento avançado de energia)

Interface que permite ao BIOS controlar as configurações do gerenciamento de energia. O APM foi substituído pela ACPI (Advanced Configuration and Power Interface, interface de energia e configuração avançada).

Advanced Technology Extended (ATX) power connector (conector de energia ATX, Advanced Technology Extended, tecnologia avançada estendida)

Conector de fonte de energia interno de 20 ou 24 pinos.

ATX (Advanced Technology Extended, tecnologia avançada estendida)

Diâmetro total de gabinete padrão para computadores modernos.

adware

Programa que exibe anúncios em um computador, geralmente distribuído com outro software.

all-in-one type printer (impressora do tipo all-in-one)

Dispositivo multifuncional desenvolvido para fornecer serviços como impressão, fax e funções de copiadora.

ANSI (American National Standards Institute, instituto nacional de padronização americano)

Organização particular e sem fins lucrativos que administra e coordena a padronização voluntária norte-americana e o sistema de avaliação de conformidade. O ANSI identifica os requisitos industriais e públicos para padrões de consenso nacionais, além de coordenar e gerenciar seu desenvolvimento, solucionar os problemas de padrões nacionais e garantir participação efetiva na padronização internacional.

ASCII (American Standard Code for Information Interchange, padrão americano de código para troca de informações)

Código de 8 bits para a representação de caracteres (7 bits mais paridade).

analog transmission (transmissão analógica)

Transmissão de sinal por cabos ou pelo ar em que as informações são transportadas através da variação de algumas combinações entre amplitude, freqüência e fase de sinal.

answer file (arquivo de respostas)

Arquivo que contém configurações predefinidas e respostas às questões exigidas pelo assistente de instalação do sistema operacional.

antistatic bag (bolsa antiestática)

Material para embalagem que protege os componentes de uma descarga eletrostática.

antistatic mat (manta antiestática)

Superfície que fornece um ambiente seguro para os componentes do computador dissipando a descarga eletrostática.

antistatic wrist strap (pulseira antiestática)

Dispositivo usado no pulso para dissipar a descarga eletrostática entre a pessoa e o equipamento eletrônico.

anti-virus application (aplicativo antivírus)

Programa instalado no sistema para evitar que o computador seja infectado por vírus.

aplicativo gráfico

Cria ou modifica imagens gráficas. Os dois tipos de imagens gráficas incluem imagens com base em vetor e objeto, além de imagens em formato bitmap ou de desenho.

AppleTalk

Conjunto de protocolos para redes com computadores Macintosh. Composto por um abrangente conjunto de protocolos que atinge as sete camadas do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos).

application layer (camada do aplicativo)

Camada 7 do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos). Essa camada fornece serviços aos processos do aplicativo, como correio eletrônico, transferência de arquivos e emulação de terminal, que estão fora do modelo OSI. A camada de aplicação identifica e estabiliza a disponibilidade dos parceiros de comunicação e os recursos necessários para conectar-se a eles, sincroniza aplicativos auxiliares e estabelece acordos sobre os procedimentos para a recuperação de erros e controle da integridade de dados. De maneira geral, corresponde às camadas de serviços de transação no modelo SNA (Systems Network Architecture, arquitetura de rede de sistemas). O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

application software (software do aplicativo)

Programa que desenvolve uma função específica aceitando a entrada do usuário e manipulando-o para atingir um resultado, conhecido como saída.

arquitetura arm (Acorn RISC Machine)

CPU com baixo índice de RISCO.

ADSL (Symmetric DSL, linha digital de assinantes simétrica)

Atualmente, é a implementação mais comum de DSL. A velocidade de downstream varia de 384 kbps para mais de 6 Mbps. Geralmente, a velocidade de upstream é menor.

asymmetric encryption (criptografia simétrica)

Método para criptografia de dados em uma rede. Usa uma chave particular para gravar mensagens e uma chave pública para decodificar as mensagens. Somente as chaves particulares precisam ser mantidas em segredo. As chaves públicas podem ser distribuídas abertamente.

attention (AT) command set (conjunto de comandos de atenção (AT) )

Emite ao modem as instruções de discagem, desligamento, redefinição, entre outras instruções. Baseia-se no conjunto de comandos de Hayes.

APIPA (Automatic Private IP Addressing, endereçamento de IP privado automático)

Recurso do sistema operacional que permite ao computador atribuir um endereço a si mesmo caso não seja possível contactar um servidor DHCP. IANA (Internet Assigned Numbers Authority) é a responsável pela alocação de endereços IP particulares para o APIPA dentro do intervalo de 169.254.0.0 a 169.254.255.255.

Automatic Update (atualização automática)

Utilitário para programar o recurso do Windows Update para verificar por atualizações críticas.

auxiliary (AUX) power connector (conector de energia auxiliar (AUX) )

Conector de 4, 6 ou 8 pinos que fornece voltagem extra para a placa-mãe a partir da fonte de energia.
