daily backup (backup diário)

Faz backup apenas dos arquivos que foram modificados no dia do backup. Este backup não redefine o bit do arquivo.

data backup (backup de dados)

Informações de um computador armazenadas em uma mídia de backup removível que pode ser mantida em local seguro. Se o hardware do computador falhar, o backup de dados pode ser restaurado para dar continuidade ao processamento.

data transfer rate (taxa de transferência de dados)

Refere-se a rapidez com que o computador pode transferir as informações para a memória.

database (banco de dados)

Coleção de dados organizada que pode ser facilmente acessada, gerenciada, indexada, pesquisada e atualizada.

data-link layer (camada de vínculo de dados)

Camada 2 do modelo de referência OSI. Essa camada permite a transitação segura dos dados por meio de um vínculo físico. A camada de vínculo de dados trata do endereçamento físico, da topologia da rede, da disciplina de linha, da notificação de erros, da entrega ordenada de quadros e do controle de fluxo. O IEEE dividiu essa camada em duas subcamadas, a subcamada MAC e a subcamada LLC. Algumas vezes, ela é chamada simplesmente de camada de vínculo. Corresponde, de maneira geral, à camada de controle de vínculo de dados do modelo SNA (Systems Network Architecture, arquitetura de rede de sistemas). O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

default gateway (gateway padrão)

Rota utilizada para que um computador em um determinado segmento possa se comunicar com um computador em outro segmento.

default installation (instalação padrão)

Instalação que requer a mínima interação com o usuário. Também conhecida por instalação típica.

default printer (impressora padrão)

Primeira opção que um aplicativo usa quando o usuário clica no ícone de impressora. A impressora padrão pode ser alterada pelo usuário.

DEFRAG

Comando que reorganiza os dados e regrava todos os arquivos no disco rígido para a inicialização da unidade, tornando mais fácil e rápida a recuperação de dados no disco rígido.

DoS (Denial of Service, ataques de negação de serviços)

Uma forma de ataque que impede aos usuários o acesso a serviços normais, como e-mail ou servidor da Web, pois o sistema fica ocupado respondendo a uma quantidade imensamente grande de solicitações. O objetivo do DoS é enviar muitas solicitações para um determinado recurso causando uma sobrecarga do sistema, cessando a operação.

desktop (área de trabalho)

Metáfora usada para representar os sistemas de arquivo. Uma área de trabalho é composta por imagens, denominadas ícones, que exibem arquivos, pastas e todos os recursos disponíveis para um usuário em um sistema operacional da GUI.

desktop computer (computadores de mesa)

Tipo de computador desenvolvido para ser usado sobre uma mesa, geralmente com monitor sobre o computador para economizar espaço. Os computadores de mesa não são portáteis como os laptops.

Device Manager (Gerenciador de dispositivos)

Aplicativo que exibe uma lista de todos os hardwares instalados no sistema.

diagnostic software (software de diagnóstico)

Programas que ajudam no processo de solução de problemas.

diagnostic tools (ferramentas de diagnóstico)

Utilitários que monitoram o servidor da rede.

DUN (dial-up networking, rede dial-up)

Usado no sistema público de telefonia ou rede para estabelecer comunicação.

differential backup (backup diferencial)

Faz backup de todos os arquivos que foram criados ou modificados desde o último backup total. Ele não redefine o bit do arquivo.

DAT (digital audio tape, fita de áudio digital)

Fita padrão que usa fitas de áudio digitais de 4 mm para armazenar dados no formato DSS (Digital Data Storage).

DLT (digital linear tape, fita linear digital)

Tecnologia que oferece recursos de backup em fita relativamente de alta velocidade e alta capacidade.

DMM (digital multimeter, multímetro digital)

Ferramenta que combina a funcionalidade de um voltímetro, ohmímetro e amperímetro em um fácil dispositivo de medição.

DSL (digital subscriber line, linha digital de assinantes)

Tecnologia de rede pública que distribui alta largura de banda por cabeamento de cobre convencional em distâncias limitadas. Tecnologia que permite para os usuários estabelecer conexão com a Internet.

DVD (digital versatile disc, disco versátil digital)

Mídia removível usada principalmente para o armazenamento de filmes e dados.

DVI (Digital Visual Interface, interface visual digital)

Interface que exibe vídeo digital não compactado em um monitor digital.

DMA (direct memory access, acesso direto à memória)

Método para ignorar a CPU ao transferir dados diretamente da memória principal para um dispositivo.

DC (direct-current, corrente contínua)

Fluxo de corrente em uma direção, como a corrente usada em uma bateria.

directory (diretório)

1) Tipo de arquivo que organiza outros arquivos em uma estrutura hierárquica. 2) Programa relacionado e arquivos de dados organizados e agrupados juntos em um sistema de arquivo DOS. 3) Usado para armazenar dados no sistema de gerenciamento de arquivos do Windows.

disk cleanup (limpeza de disco)

Software de gerenciamento de disco usado para limpar espaço em um disco rígido buscando por arquivos que podem ser seguramente excluídos, como os arquivos temporários da Internet.

disk management (gerenciamento de disco)

Utilitário do sistema usado para gerenciar discos rígidos e partições, como discos de inicialização, criação e formatação de partições.

DOS (disk operating system, sistema operacional em disco)

Conjunto de programas e comandos que controlam as operações gerais do computador em um sistema baseado em disco.

display (monitor)

Mecanismo de projeção e saída do computador que mostra texto e imagens gráficas.

DNS poisoning (Inviabilização de DNS)

Alteração nos registros DNS em um sistema que visa apontar falsos servidores em que os dados estão registrados.

docking connector (conector multifuncional)

Soquete usado para conectar uma base multifuncional ao laptop.

docking station (base multifuncional)

Dispositivo que conecta um laptop a uma corrente alternada e periféricos do computador de mesa.

domain (domínio)

Grupo lógico de computadores e dispositivos eletrônicos com um conjunto de regras e procedimentos comuns administrados como uma unidade.

DNS (Domain Name System, sistema de nomes de domínios)

Sistema que fornece uma maneira de mapear amigavelmente nomes de host, URLs ou endereços IP.

dot matrix printer (impressora matricial)

Impressora que usa a técnica de prensar uma tira contra o papel para exibir a impressão.

dpi (dots per inch, pontos por polegada)

Medida da qualidade da impressão. Como a qualidade da impressão é medida em uma impressora matricial, quanto maior o dpi, maior a qualidade.

drive bay (compartimento da unidade)

Área de tamanho padrão para adicionar hardware ao gabinete do computador. Os dois compartimentos de unidade mais comuns são a unidade de CD\DVD e a unidade de disquete.

drive letter (letra da unidade)

Designação que distingue as unidades físicas ou lógicas no Windows.

drive mapping (mapeamento de unidade)

Processo de atribuição de uma letra a uma unidade física ou lógica.

DIMM (dual in-line memory module, modulo de memória em linha dupla)

Placa de circuitos com barramento de dados de 64 bits que retém os chips de memória. Módulo de memória com 168 pinos. Oferece suporte a transferências de dados de 64 bits.

dual ring (duplo anel)

Todos os dispositivos da rede são conectados a dois cabos e os dados são transportados em ambas as direções. Apenas um cabo é usado por vez. Caso ocorra uma falha em um dos cabos/anéis, os dados são transmitidos pelo outro cabo/anel.

DVD drive (unidade de DVD)

Dispositivo óptico que lê DVDs. Uma unidade de DVD regravável (DVD-RW) permite realizar gravações em DVD-RWs.

DVD-ROM

Formato de DVD desenvolvido para armazenar arquivos do computador.

DVD-RW

Tecnologia que permite que a mídia seja gravada várias vezes.

DVD-Video (Vídeo de DVD)

Formato de DVD usado por DVD players independentes para filmes e extras.

dye-sublimation printer (impressoras dye-sublimation)

Impressora que usa camadas de tinta sólida que se transformam em gasosas em um processo chamado sublimação. O gás, então, passa para o papel e volta para o estado sólido. O cabeçote de impressão aplica uma camada de ciano, magenta, amarelo e uma cobertura clara (CMYO). Também é chamada de impressora de transferência térmica.

DHCP (Dynamic Host Configuration Protocol, protocolo de configuração dinâmica de hosts)

Utilitário que atribui automaticamente os endereços IP aos dispositivos do cliente em uma grande rede.

DRAM (dynamic RAM, RAM dinâmica)

RAM que armazena informações em condensadores que devem ser atualizados periodicamente. Os atrasos podem ocorrer porque as DRAMs não podem ser acessadas pelo processador no momento da atualização de seu conteúdo. Entretanto, as DRAMs são menos complexas e possuem mais recursos que as SRAMs.

dynamic routing (roteamento dinâmico)

Roteamento que ajusta automaticamente a topologia da rede ou as alterações de tráfego. Também chamado de roteamento adaptativo.
