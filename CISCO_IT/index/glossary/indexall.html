AC power connector (conector de energia CA)

Soquete usado para conectar o adaptador de energia CA a um computador ou base multifuncional.

AC power cord (cabo de energia CA)

Cabo que transfere eletricidade da fonte de energia CA para a fonte de energia do computador.

AGP (Accelerated Graphics Port, porta de gráficos acelerada)

Tecnologia de barramento de 32 bits, de alta velocidade desenvolvida para oferecer suporte à aceleração de gráficos em 3D no computador.

ACL (Access Control List, lista de controle de acesso)

Lista gerenciada por um administrador de rede que registra os detalhes sobre o que um usuário pode acessar e o tipo de acesso concedido.

access point (ponto de acesso)

Dispositivo que conecta dispositivos sem fio para formar uma rede sem fio. Um ponto de acesso geralmente é conectado a uma rede cabeada e pode restabelecer os dados entre dispositivos com e sem fio. As distâncias de conectividade podem ser de vários pés ou metros a várias milhas ou quilômetros.

active partition (partição ativa)

Partição em uma unidade de disco rígido que é definida como a partição inicializável e normalmente contém o sistema operacional a ser usado no computador. No computador, apenas uma partição pode ser definida como partição inicializável ou ativa na unidade de disco rígido.

ActiveX

Applet ou pequeno programa criado pela Microsoft para controlar a interatividade nas páginas da Web que devem ser baixadas para obter acesso a todas as funções.

adapter card (placa do adaptador)

Placa de expansão que aumenta o número de controladores e portas disponíveis em um computador.

ARP (Address Resolution Protocol, protocolo de resolução de endereços)

Descobre o endereço local (endereço MAC) de uma estação na rede quando o endereço IP é desconhecido. Estações finais e roteadores usam o ARP para descobrir endereços locais:
-a Switch usado com o comando ARP que exibe o cache.
-d Switch usado com o comando ARP que exclui uma entrada do cache ARP.
-s Switch usado com o comando ARP que adiciona um mapeamento permanente de endereço IP para MAC.

administrator (administrador)

Pessoa que consulta o Registro de Usuário para analisar o status de assinante individual e para coletar dados.

ACPI (Advanced Configuration and Power Interface, interface de energia e configuração avançada)

Interface que permite ao sistema operacional controlar o gerenciamento de energia. Substitui o APM (Advanced Power Management, gerenciamento avançado de energia).

APM (Advanced Power Management, gerenciamento avançado de energia)

Interface que permite ao BIOS controlar as configurações do gerenciamento de energia. O APM foi substituído pela ACPI (Advanced Configuration and Power Interface, interface de energia e configuração avançada).

Advanced Technology Extended (ATX) power connector (conector de energia ATX, Advanced Technology Extended, tecnologia avançada estendida)

Conector de fonte de energia interno de 20 ou 24 pinos.

ATX (Advanced Technology Extended, tecnologia avançada estendida)

Diâmetro total de gabinete padrão para computadores modernos.

adware

Programa que exibe anúncios em um computador, geralmente distribuído com outro software.

all-in-one type printer (impressora do tipo all-in-one)

Dispositivo multifuncional desenvolvido para fornecer serviços como impressão, fax e funções de copiadora.

ANSI (American National Standards Institute, instituto nacional de padronização americano)

Organização particular e sem fins lucrativos que administra e coordena a padronização voluntária norte-americana e o sistema de avaliação de conformidade. O ANSI identifica os requisitos industriais e públicos para padrões de consenso nacionais, além de coordenar e gerenciar seu desenvolvimento, solucionar os problemas de padrões nacionais e garantir participação efetiva na padronização internacional.

ASCII (American Standard Code for Information Interchange, padrão americano de código para troca de informações)

Código de 8 bits para a representação de caracteres (7 bits mais paridade).

analog transmission (transmissão analógica)

Transmissão de sinal por cabos ou pelo ar em que as informações são transportadas através da variação de algumas combinações entre amplitude, freqüência e fase de sinal.

answer file (arquivo de resposta)

Arquivo que contém configurações predefinidas e respostas às questões exigidas pelo assistente de instalação do sistema operacional.

antistatic bag (bolsa antiestática)

Material para embalagem que protege os componentes de uma descarga eletrostática.

antistatic mat (manta antiestática)

Superfície que fornece um ambiente seguro para os componentes do computador dissipando a descarga eletrostática.

antistatic wrist strap (pulseira antiestática)

Dispositivo usado no pulso para dissipar a descarga eletrostática entre a pessoa e o equipamento eletrônico.

anti-virus application (aplicativo antivírus)

Programa instalado no sistema para evitar que o computador seja infectado por vírus.

aplicativo gráfico

Cria ou modifica imagens gráficas. Os dois tipos de imagens gráficas incluem imagens com base em vetor e objeto, além de imagens em formato bitmap ou de desenho.

AppleTalk

Conjunto de protocolos para redes com computadores Macintosh. Composto por um abrangente conjunto de protocolos que atinge as sete camadas do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos).

application layer (camada de aplicação)

Camada 7 do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos). Essa camada fornece serviços aos processos do aplicativo, como correio eletrônico, transferência de arquivos e emulação de terminal, que estão fora do modelo OSI. A camada de aplicação identifica e estabiliza a disponibilidade dos parceiros de comunicação e os recursos necessários para conectar-se a eles, sincroniza aplicativos auxiliares e estabelece acordos sobre os procedimentos para a recuperação de erros e controle da integridade de dados. De maneira geral, corresponde às camadas de serviços de transação no modelo SNA (Systems Network Architecture, arquitetura de rede de sistemas). O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

application software (software aplicativo)

Programa que desenvolve uma função específica aceitando a entrada do usuário e manipulando-o para atingir um resultado, conhecido como saída.

arm (Acorn RISC Machine) architecture (arquitetura arm, Acorn RISC Machine)

CPU com baixo índice de RISCO.

ADSL (Asymmetric DSL, linha digital de assinantes assimétrica)

Atualmente, é a implementação mais comum de DSL. A velocidade de downstream varia de 384 kbps para mais de 6 Mbps. Geralmente, a velocidade de upstream é menor.

asymmetric encryption (criptografia assimétrica)

Método para criptografia de dados em uma rede. Usa uma chave particular para gravar mensagens e uma chave pública para decodificar as mensagens. Somente as chaves particulares precisam ser mantidas em segredo. As chaves públicas podem ser distribuídas abertamente.

attention (AT) command set (conjunto de comandos de atenção (AT) )

Emite ao modem as instruções de discagem, desligamento, redefinição, entre outras instruções. Baseia-se no conjunto de comandos de Hayes.

APIPA (Automatic Private IP Addressing)

Recurso do sistema operacional que permite ao computador atribuir um endereço a si mesmo caso não seja possível contactar um servidor DHCP. IANA (Internet Assigned Numbers Authority) é a responsável pela alocação de endereços IP particulares para o APIPA dentro do intervalo de 169.254.0.0 a 169.254.255.255.

Automatic Update (atualização automática)

Utilitário para programar o recurso do Windows Update para verificar por atualizações críticas.

auxiliary (AUX) power connector (conector de energia auxiliar (AUX))

Conector de 4, 6 ou 8 pinos que fornece voltagem extra para a placa-mãe a partir da fonte de energia.

backplane

Conexão física entre uma placa ou processador de interface, barramentos de dados e barramentos de distribuição de energia dentro de um chassi.

backup

Cópia de dados salva em uma mídia alternativa e deve ser fisicamente removida dos dados de origem.

backward compatible (compatível com versões anteriores)

Sistema de hardware ou software que pode usar interfaces e dados a partir de versões anteriores do sistema ou com outros sistemas. Também conhecido como 'compatível com versão anterior' ou 'backward compatible'.

bandwidth (largura de banda)

Quantidade de dados que pode ser transmitida em um determinado período de tempo.

base station (estação de base)

Dispositivo que conecta um laptop a uma corrente alternada e periféricos do computador de mesa.

BIOS (Basic Input/Output System, sistema de entrada e saída básico)

Programa armazenado em um chip de ROM que fornece o código básico para controlar o hardware do computador e para desenvolver diagnósticos sobre ele. O BIOS prepara o computador para carregar o sistema operacional.

BRI (Basic Rate Interface, interface de taxa básica)

Interface ISDN (Integrated Services Digital Network, rede digital de serviços integrados) composta por dois canais B e um canal D para comunicação comutada por circuitos de voz, vídeo e dados. Compare com PRI.

battery latch (trava da bateria)

Ferramenta usada para inserir, remover e segurar a bateria do laptop.

battery (bateria)

Dispositivo elétrico que converte energia química em energia elétrica.

beep code (código de bipes)

Sistema sonoro de relatórios sobre os erros que são encontrados no BIOS durante o POST, representado por uma série de bipes.

Berg power connector (conector de energia Berg)

Conector chaveado que fornece energia a uma unidade de disquete.

biometric device (dispositivo biométrico)

Ferramenta que usa sensores, como impressão digital ou scanner de retina, para identificar as características físicas do usuário e permitir seu acesso a um dispositivo ou uma rede.

bit rate (taxa de bits)

Velocidade em que os bits são transmitidos, normalmente expressos em bits por segundo (bps).

bit

Menor unidade de dados em um computador. Um bit pode ser definido como 1 ou 0. O bit é o formato binário em que os dados são processados pelos computadores.

blackout (blecaute)

Perda total de energia CA.

Bluetooth

Padrão industrial sem fio que usa uma freqüência de rádio não licenciada para estabelecer comunicações em curtas distâncias permitindo que os dispositivos portáteis possam se comunicar em longas distâncias.

boot (inicialização)

Para iniciar um computador.

boot record (registro de inicialização)

Arquivo de 512 bytes contendo uma tabela que descreve a partição, o número de bytes por setor e o número de setores por cluster.

bootable disk (disco inicializável)

Ferramenta de solução de problemas que permite inicializar o computador a partir de um disco quando o disco rígido não é capaz de inicializá-lo.

broadband (banda larga)

Vários sinais usando diversas freqüências em um cabo.

broadband optical telepoint (teleponto óptico de banda larga)

Transmissão de banda larga por infravermelho capaz de atender aos requisitos de multimídia de alta qualidade.

broadband satellite (banda larga por satélite)

Conexão de rede usando uma antena parabólica.

brownout (queda de energia)

Interrupção temporária de energia CA.

buffer

Área de armazenamento usada para manipular dados em trânsito. Os buffers são usados em interligações de rede para compensar as diferenças na velocidade do processamento entre os dispositivos de rede. O excesso de dados pode ser armazenado em buffers até que eles sejam manipulados por dispositivos de processamento mais lentos. Algumas vezes são citados como um buffers de pacote.

bus topology (topologia de barramento)

Rede com computadores conectados a um cabo comum.

bus (barramento)

Meio através da qual os dados são transferidos de uma parte a outra do computador. O barramento pode ser comparado a uma highway, onde os dados literalmente viajam dentro do computador.

byte

Uma unidade de medida que descreve o tamanho de um arquivo de dados, a quantidade de espaço em um disco, ou outro meio de armazenamento, ou a quantidade de dados que está sendo enviada através da rede. Um byte é composto por 8 bits de dados.

C: drive (unidade C)

Geralmente indica a primeira unidade de disco rígido em um sistema de computador. A unidade A e B são reservadas para unidades de disquetes. A unidade B raramente é usada nos computadores atuais.

CA (alternating-current, corrente alternada)

Corrente que altera a direção em uma taxa de repetição uniforme. Geralmente, esse tipo de eletricidade é fornecido por uma concessionária de serviços públicos e pode ser obtido por tomadas de parede.

cable modem

Age como uma interface LAN conectando o computador com a Internet. O cable modem conecta um computador à rede a cabo da empresa através do mesmo cabo coaxial que alimenta os sinais de TV a cabo (CATV) em uma televisão.

cable tie (amarração de cabos)

Prendedor para unir os cabos dentro e fora do computador.

cable (cabo)

Conjunto de condutores amarrados todos juntos, confeccionados em cobre ou fibra óptica que transportam sinais e energia aos dispositivos elétricos.

cache

Área de armazenamento de dados que fornece acesso de alta velocidade para o sistema.

Caps lock indicator LED (LED indicador de Caps Lock)

Luz que indica se a tecla Caps Lock está ativada ou desativada.

card key (chave do cartão)

Cartão de identidade com chip que armazena dados de usuário, inclusive o nível de acesso.

Category 3 (Categoria 3)

Cabo usado principalmente em conexões telefônicas.

Category 5 (Categoria 5)

Cabo que contém quatro pares de fio, com uma taxa de dados máxima de 1 Gbps.

Category 5e (Categoria 5e)

Cabo que fornece mais pares trançados por pés do que o cabo Categoria 5 na mesma taxa de dados de 1 Gbps.

Category 6 (Categoria 6)

Cabo com mais pares trançados do que o cabo Categoria 5e. Ele contém um divisor plástico que separa os pares de fios evitando interferências.

cellular WAN (WAN para celular)

Rede de longa distância cuja tecnologia permite o uso de um telefone celular ou laptop para estabelecer comunicações por dados e voz.

CPU (central processing unit, unidade central de processamento)

Interpreta e processa instruções e dados do software. Localizado na placa-mãe, a CPU é um chip contido em um circuito integrado simples chamado microprocessador. A CPU contém dois componentes básicos, uma unidade de controle e uma ALU (Arithmetic/Logic Unit, unidade lógica e aritmética).

CPU dual core

Dois núcleos dentro de um único chip da CPU. Ambos os núcleos podem ser usados em conjunto para aumentar a velocidade ou podem ser usados em dois locais ao mesmo tempo.

chip

Pequena peça de silicone ou germânio processada para ter características elétricas de forma que possa ser desenvolvida em um componente eletrônico. Também chamada de semicondutor.

chip set (chipset)

Conjunto de chips em uma placa-mãe que permite à CPU comunicar-se e interagir com outros componentes do computador.

CHKDSK

Comando usado para verificar a integridade de arquivos e pastas no disco rígido examinando a superfície do disco em busca de erros físicos.

client/server network (rede cliente/servidor)

Rede em que os serviços estão localizados em um computador dedicado que responde às solicitações do cliente ou do usuário.

cluster

Menor unidade de espaço usada para armazenar dados em um disco. Também chamada de unidade de alocação de arquivo.

CMOS battery (bateria CMOS)

Bateria que fornece energia para manter as informações de configurações básicas, incluindo o relógio em tempo real, quando o computador é desligado.

CMYK

Exibe as cores: ciano, magenta, amarelo e preto.

coaxial cable (cabo coaxial)

Cabo blindado com revestimento de cobre usado para conectar computadores em uma rede.

cold boot (inicialização com computador desligado)

Para ligar um computador desligado.

color ink jet printer (impressora jato de tinta colorida)

Tipo de impressora que usa cartucho de tinta líquida, borrifando a tinta para formar uma imagem no papel.

CCITT (Comité Consultatif International Téléphonique et Télégraphique, comitê consultivo telefônico e telegráfico internacional)

Comitê que define os padrões internacionais de comunicação. O CCITT define os padrões para o envio de documentos via fax e os padrões para a transmissão de dados em linhas telefônicas.

CLI (command line interface, interface da linha de comando)

Interface, como, por exemplo, o prompt do DOS, que requer que os comandos sejam inseridos manualmente na linha de comando.

CD-ROM (compact disc - read only memory)

Mídia óptica de armazenamento para áudio e dados.

compact disc (CD) drive (unidade de CD (compact disc) )

Dispositivo óptico que lê CDs (discos compactos).

CD-R (compact disc-recordable, disco compacto gravável)

Mídia óptica que permite que os dados sejam gravados, mas não modificados.

CD-RW (compact disc-rewritable, disco compacto regravável)

Mídia óptica de armazenamento que permite que os dados sejam gravados e modificados.

CMOS (Complementary Metal Oxide Semiconductor, semicondutor complementar de óxido metálico)

Tipo de semicondutor, ou firmware de memória de baixa energia, que armazena as informações de configurações básicas.

CISC (Complex Instruction Set Computer, computador com conjunto de instruções complexo)

Arquitetura que usa um amplo conjunto de instruções com várias opções para quase toda operação. O resultado é que um programador pode executar precisamente o comando necessário, resultando em menos etapas de instrução por operação.

compressed air (ar comprimido)

Lata de ar pressurizado usada para remover a poeira dos componentes do computador sem criar estática. Também chamado de ar enlatado.

computer (computador)

Máquina elétrica que pode executar uma lista de instruções e desenvolver cálculos com base nessas instruções.

computer network (rede de computadores)

Dois ou mais computadores conectados em conjunto por algum tipo de mídia para compartilhar dados e recursos.

computer system (sistema de computador)

Combinação de componentes de hardware e software. Hardware é o equipamento físico, como gabinetes, unidades de disquetes, teclados, monitores, cabos, alto-falantes e impressoras. O software descreve os programas que operam em um sistema de computador.

CAD (computer-aided design)

Aplicativo usado para a criação de projetos arquitetônicos, elétricos e mecânicos. As formas mais complexas do CAD incluem modelagens de paramétricas e sólidas, que permitem que os objetos sejam criados com características reais.

conduit (conduíte)

Proteção para a mídia de infra-estrutura contra danos e acesso não autorizado.

configuration tool (ferramenta de configuração)

Ferramenta de gerenciamento de serviço ou uma ferramenta de serviço de gerenciamento de Elemento com uma GUI.

connector (conector)

Dispositivo usado no final de um cabo.

conventional memory (memória convencional)

Todos os endereços de memória de 0 a 640 KB.

cookie

Pequeno arquivo de texto armazenado no disco rígido que permite a um site controlar a associação do usuário a esse site.

copy backup (cópia de backup)

Faz backup em fita dos arquivos selecionados pelo usuário. Este backup não redefine o bit do arquivo.

crosstalk (diafonia)

Interferência de energia, como, por exemplo, interferência eletromagnética (EMI), que é transferida de um circuito para outro.

current (I) (corrente (I) )

Fluxo de elétrons em um condutor medido em amperes.

CRU (customer replaceable unit, unidade substituível pelo cliente)

Componente que pode ser instalado pelo usuário.

cylinder (cilindro)

Todas as faixas em um disco rígido com o mesmo número. Coletivamente, a mesma faixa em todos os discos de um disco rígido multidisco.

daily backup (backup diário)

Faz backup apenas dos arquivos que foram modificados no dia do backup. Este backup não redefine o bit do arquivo.

data backup (backup de dados)

Informações de um computador armazenadas em uma mídia de backup removível que pode ser mantida em local seguro. Se o hardware do computador falhar, o backup de dados pode ser restaurado para dar continuidade ao processamento.

data transfer rate (taxa de transferência de dados)

Refere-se a rapidez com que o computador pode transferir as informações para a memória.

database (banco de dados)

Coleção de dados organizada que pode ser facilmente acessada, gerenciada, indexada, pesquisada e atualizada.

data-link layer (camada de vínculo de dados)

Camada 2 do modelo de referência OSI. Essa camada permite a transitação segura dos dados por meio de um vínculo físico. A camada de vínculo de dados trata do endereçamento físico, da topologia da rede, da disciplina de linha, da notificação de erros, da entrega ordenada de quadros e do controle de fluxo. O IEEE dividiu essa camada em duas subcamadas, a subcamada MAC e a subcamada LLC. Algumas vezes, ela é chamada simplesmente de camada de vínculo. Corresponde, de maneira geral, à camada de controle de vínculo de dados do modelo SNA (Systems Network Architecture, arquitetura de rede de sistemas). O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

default gateway (gateway padrão)

Rota utilizada para que um computador em um determinado segmento possa se comunicar com um computador em outro segmento.

default installation (instalação padrão)

Instalação que requer a mínima interação com o usuário. Também conhecida por instalação típica.

default printer (impressora padrão)

Primeira opção que um aplicativo usa quando o usuário clica no ícone de impressora. A impressora padrão pode ser alterada pelo usuário.

DEFRAG

Comando que reorganiza os dados e regrava todos os arquivos no disco rígido para a inicialização da unidade, tornando mais fácil e rápida a recuperação de dados no disco rígido.

DoS (Denial of Service, ataques de negação de serviços)

Uma forma de ataque que impede aos usuários o acesso a serviços normais, como e-mail ou servidor da Web, pois o sistema fica ocupado respondendo a uma quantidade imensamente grande de solicitações. O objetivo do DoS é enviar muitas solicitações para um determinado recurso causando uma sobrecarga do sistema, cessando a operação.

desktop (área de trabalho)

Metáfora usada para representar os sistemas de arquivo. Uma área de trabalho é composta por imagens, denominadas ícones, que exibem arquivos, pastas e todos os recursos disponíveis para um usuário em um sistema operacional da GUI.

desktop computer (computadores de mesa)

Tipo de computador desenvolvido para ser usado sobre uma mesa, geralmente com monitor sobre o computador para economizar espaço. Os computadores de mesa não são portáteis como os laptops.

Device Manager (Gerenciador de dispositivos)

Aplicativo que exibe uma lista de todos os hardwares instalados no sistema.

diagnostic software (software de diagnóstico)

Programas que ajudam no processo de solução de problemas.

diagnostic tools (ferramentas de diagnóstico)

Utilitários que monitoram o servidor da rede.

DUN (dial-up networking, rede dial-up)

Usado no sistema público de telefonia ou rede para estabelecer comunicação.

differential backup (backup diferencial)

Faz backup de todos os arquivos que foram criados ou modificados desde o último backup total. Ele não redefine o bit do arquivo.

DAT (digital audio tape, fita de áudio digital)

Fita padrão que usa fitas de áudio digitais de 4 mm para armazenar dados no formato DSS (Digital Data Storage).

DLT (digital linear tape, fita linear digital)

Tecnologia que oferece recursos de backup em fita relativamente de alta velocidade e alta capacidade.

DMM (digital multimeter, multímetro digital)

Ferramenta que combina a funcionalidade de um voltímetro, ohmímetro e amperímetro em um fácil dispositivo de medição.

DSL (digital subscriber line, linha digital de assinantes)

Tecnologia de rede pública que distribui alta largura de banda por cabeamento de cobre convencional em distâncias limitadas. Tecnologia que permite para os usuários estabelecer conexão com a Internet.

DVD (digital versatile disc, disco versátil digital)

Mídia removível usada principalmente para o armazenamento de filmes e dados.

DVI (Digital Visual Interface, interface visual digital)

Interface que exibe vídeo digital não compactado em um monitor digital.

DMA (direct memory access, acesso direto à memória)

Método para ignorar a CPU ao transferir dados diretamente da memória principal para um dispositivo.

DC (direct-current, corrente contínua)

Fluxo de corrente em uma direção, como a corrente usada em uma bateria.

directory (diretório)

1) Tipo de arquivo que organiza outros arquivos em uma estrutura hierárquica. 2) Programa relacionado e arquivos de dados organizados e agrupados juntos em um sistema de arquivo DOS. 3) Usado para armazenar dados no sistema de gerenciamento de arquivos do Windows.

disk cleanup (limpeza de disco)

Software de gerenciamento de disco usado para limpar espaço em um disco rígido buscando por arquivos que podem ser seguramente excluídos, como os arquivos temporários da Internet.

disk management (gerenciamento de disco)

Utilitário do sistema usado para gerenciar discos rígidos e partições, como discos de inicialização, criação e formatação de partições.

DOS (disk operating system, sistema operacional em disco)

Conjunto de programas e comandos que controlam as operações gerais do computador em um sistema baseado em disco.

display (monitor)

Mecanismo de projeção e saída do computador que mostra texto e imagens gráficas.

DNS poisoning (Inviabilização de DNS)

Alteração nos registros DNS em um sistema que visa apontar falsos servidores em que os dados estão registrados.

docking connector (conector multifuncional)

Soquete usado para conectar uma base multifuncional ao laptop.

docking station (base multifuncional)

Dispositivo que conecta um laptop a uma corrente alternada e periféricos do computador de mesa.

domain (domínio)

Grupo lógico de computadores e dispositivos eletrônicos com um conjunto de regras e procedimentos comuns administrados como uma unidade.

DNS (Domain Name System, sistema de nomes de domínios)

Sistema que fornece uma maneira de mapear amigavelmente nomes de host, URLs ou endereços IP.

dot matrix printer (impressora matricial)

Impressora que usa a técnica de prensar uma tira contra o papel para exibir a impressão.

dpi (dots per inch, pontos por polegada)

Medida da qualidade da impressão. Como a qualidade da impressão é medida em uma impressora matricial, quanto maior o dpi, maior a qualidade.

drive bay (compartimento da unidade)

Área de tamanho padrão para adicionar hardware ao gabinete do computador. Os dois compartimentos de unidade mais comuns são a unidade de CD\DVD e a unidade de disquete.

drive letter (letra da unidade)

Designação que distingue as unidades físicas ou lógicas no Windows.

drive mapping (mapeamento de unidade)

Processo de atribuição de uma letra a uma unidade física ou lógica.

DIMM (dual in-line memory module, modulo de memória em linha dupla)

Placa de circuitos com barramento de dados de 64 bits que retém os chips de memória. Módulo de memória com 168 pinos. Oferece suporte a transferências de dados de 64 bits.

dual ring (duplo anel)

Todos os dispositivos da rede são conectados a dois cabos e os dados são transportados em ambas as direções. Apenas um cabo é usado por vez. Caso ocorra uma falha em um dos cabos/anéis, os dados são transmitidos pelo outro cabo/anel.

DVD drive (unidade de DVD)

Dispositivo óptico que lê DVDs. Uma unidade de DVD regravável (DVD-RW) permite realizar gravações em DVD-RWs.

DVD-ROM

Formato de DVD desenvolvido para armazenar arquivos do computador.

DVD-RW

Tecnologia que permite que a mídia seja gravada várias vezes.

DVD-Video (Vídeo de DVD)

Formato de DVD usado por DVD players independentes para filmes e extras.

dye-sublimation printer (impressoras dye-sublimation)

Impressora que usa camadas de tinta sólida que se transformam em gasosas em um processo chamado sublimação. O gás, então, passa para o papel e volta para o estado sólido. O cabeçote de impressão aplica uma camada de ciano, magenta, amarelo e uma cobertura clara (CMYO). Também é chamada de impressora de transferência térmica.

DHCP (Dynamic Host Configuration Protocol, protocolo de configuração dinâmica de hosts)

Utilitário que atribui automaticamente os endereços IP aos dispositivos do cliente em uma grande rede.

DRAM (dynamic RAM, RAM dinâmica)

RAM que armazena informações em condensadores que devem ser atualizados periodicamente. Os atrasos podem ocorrer porque as DRAMs não podem ser acessadas pelo processador no momento da atualização de seu conteúdo. Entretanto, as DRAMs são menos complexas e possuem mais recursos que as SRAMs.

dynamic routing (roteamento dinâmico)

Roteamento que ajusta automaticamente a topologia da rede ou as alterações de tráfego. Também chamado de roteamento adaptativo.

eject button (botão ejetar)

Libera um objeto, como o botão em uma unidade de disquete.

EIA (Electronic Industries Association, associação das indústrias eletrônicas)

Grupo que especifica os padrões de transmissão elétrica. A EIA e o TIA desenvolveram vários padrões de comunicação bastante conhecidos, incluindo o EIA/TIA-232 e o EIA/TIA-449.

e-mail (correio eletrônico)

Habilidade que permite aos usuários se comunicarem por uma rede de computador. Troca de mensagens armazenadas no computador por meio de uma comunicação via rede.

electrophotographic drum (tambor eletrofotográfico)

Parte central da impressora a laser que usa o toner para imprimir no papel.

ESD (electrostatic discharge, descarga eletrostática)

Descarga de eletricidade estática de um condutor para outro de potências diferentes.

encryption (criptografia)

Recurso de segurança que aplica uma codificação em um arquivo de forma que apenas os usuários autorizados possam visualizar o arquivo.

EFS (Encryption File System, criptografia de sistema de arquivos)

Sistema de arquivos para criptografia específico da Microsoft.

EIDE (Enhanced Integrated Drive Electronics)

Versão aprimorada da interface IDE padrão que conecta discos rígidos, unidades de CD-ROM e unidades de fita a um PC.

Ethernet

Especificação de LAN de banda base criada pela Xerox Corporation e desenvolvida em conjunto pela Xerox, Intel e pela Digital Equipment Corporation. As redes Ethernet usam o CSMA/CD e são executadas por uma variedade de tipos de cabos a 10 Mbps ou mais. A Ethernet é semelhante com a série de padrões IEEE 802.3.

Ethernet port (porta Ethernet)

Soquete RJ-45 usado para conectar um computador a uma rede de área local (LAN) por cabo.

event (evento)

Mensagem da rede que indica irregularidades operacionais nos elementos físicos de uma rede ou uma resposta para a ocorrência de uma tarefa significativa, geralmente a conclusão de uma solicitação de informações.

Event Viewer (Visualizar eventos)

Aplicativo que monitora eventos do sistema, eventos do aplicativo e eventos de segurança.

exhaust vent (aberturas de exaustão)

Saída que libera ar quente do interior de um dispositivo ou sala.

expansion card modem (modem da placa de expansão)

Modem inserido no slot de expansão da placa-mãe (ISA ou PCI). Também chamado de modem interno.

expansion slot (slot de expansão)

Abertura no computador que permite a inserção de uma placa de PC para adicionar recursos ao computador.

ExpressCard

Placa padrão de expansão de laptop e de alto-processamento desenvolvida pela PCMCIA. O slot de expansão ExpressCard usa a PCI Express (x1) incorporada e/ou o barramento USB de um laptop. As ExpressCards possuem um conector de 26 pinos e são intercambiáveis.

extended memory (memória estendida)

Memória superior a 1 MB.

extended partition (partição estendida)

Segunda partição no disco rígido.

extended-star topology (topologia estrela estendida)

Topologia estrela expandida para incluir dispositivos de rede adicionais.

external hard drive (disco rígido externo)

Dispositivo que é conectado ao computador para fornecer armazenamento de dados adicional.

external modem (modem externo)

Modem que é conectado à porta serial (COM1 ou COM2) da maioria dos computadores. Um modem externo, como um cable modem, normalmente é usado em conexões de alta velocidade.

Fast Ethernet

Qualquer uma das especificações de Ethernet de 100 Mbps. A Fast Ethernet oferece velocidade dez vezes maior que a especificação 10BASE-T Ethernet, preservando algumas qualidades como formato de quadros, mecanismos MAC e MTU. Tais semelhanças permitem o uso de aplicativos 10BASE-T existentes e ferramentas de gerenciamento de rede em redes Fast Ethernet. Baseia-se na extensão da especificação IEEE 802.3. Compare com Ethernet.

FDISK

Comando usado para excluir e criar partições em disco rígido:\STATUS; switch que exibe as informações de partição quando utilizadas com o comando FDISK.

FDDI (Fiber Distributed Data Interface, interface de dados distribuídos em fibra óptica)

Tipo de rede Token Ring usada em grandes LANs.

fiber-optic cable (cabo de fibra óptica)

Meio físico capaz de conduzir a transmissão de iluminação modulada. Se comparado com outro meio de transmissão, o cabo de fibra óptica é mais caro, mas não está sujeito a interferência eletromagnética e é capaz de trabalhar com taxas de dados mais altas. Algumas vezes, é chamado de fibra óptica.

fiber-optic cable (fibra óptica)

Usa cabos com base em vidro ou plástico, também chamados de fibra, para conduzir informações como pulsos de iluminação. Conduz a iluminação modulada para transmitir dados.

FRU (field-replaceable unit, unidade substituível em campo)

Componente que um técnico qualificado pode instalar em um local remoto.

file (arquivo)

Um bloco de dados relacionados logicamente que recebe um único nome e é tratado como uma única unidade.

FAT (file allocation table, tabela de alocação de arquivos)

Tabela usada pelo sistema operacional para armazenar informações sobre o local dos arquivos armazenados em um disco. Esse arquivo é armazenado em uma faixa 0 no disco.

file extension (extensão de arquivo)

Designação que descreve o formato do arquivo ou o tipo de aplicativo que criou um arquivo.

file management (gerenciamento de arquivos)

Estrutura hierárquica de arquivos, pastas e unidades no Windows.

file system (sistema de arquivo)

Os dois sistemas de arquivos disponíveis no Windows XP são FAT32 e NTFS. NTFS apresenta maior estabilidade e recursos de segurança.

FTP (File Transfer Protocol, protocolo de transferência de arquivos)

Conjunto de regras que rege como os arquivos são transferidos. O FTP permite várias conexões simultâneas a sistemas de arquivos remotos.

fingerprint reader (leitor de impressão digital)

Dispositivo de entrada que examina as impressões digitais para autenticar o login usando uma identificação biométrica.

firewall

Roteador ou servidor de acesso, ou vários roteadores e servidores de acesso, designado como um buffer entre todas as redes públicas conectadas e uma rede particular. Um roteador de firewall usa listas de acessos e outros métodos para garantir a segurança da rede particular.

FireWire

Barramento de comunicação de plataforma independente de alta velocidade. O FireWire permite a interconexão de dispositivos digitais, como câmeras de vídeo digitais, impressoras, scanners, câmeras digitais e discos rígidos. O FireWire também é conhecido como IEEE 1394, i.Link (propriedade da Sony), e LHDC (linear heat detecting cable) no Reino Unido.

firmware

Programa incorporado em um chip de silicone em vez de armazenado em uma unidade de disquete.

Flash memory (memória Flash)

Chip de memória regravável que retém os dados após o desligamento da energia.

flat-head screwdriver (chave de fenda)

Ferramenta usada para soltar ou apertar os parafusos dos slots.

floppy data cable (cabo de dados do disquete)

Cabo externo que transfere dados entre o computador e a unidade de disquete.

FDD (floppy disk drive, unidade de disquete)

Dispositivo com disco magnético giratório que permite a leitura e a gravação de dados.

floppy drive cable (cabo da unidade de disquete)

Cabo externo que conecta o computador e a unidade de disquete.

form factor (diâmetro total)

Tamanho físico e forma dos componentes do computador. Os componentes que compartilham o mesmo diâmetro total são fisicamente intercambiáveis.

format (formato)

Para preparar um sistema de arquivos em uma partição com o objetivo de armazenar arquivos.

full backup (backup total)

Faz backup de todos os arquivos em um disco. Também chamado de backup normal.

full-duplex transmission (transmissão full-duplex)

Transmissão de dados que pode ser conduzida de duas formas ao mesmo tempo. Uma conexão com a Internet que usa o serviço DSL é um exemplo.

function key (tecla de função (tecla Fn) )

Tecla modificadora geralmente encontrada em laptops. É usada em conjunto com outras teclas para desenvolver funções específicas.

gamepad

Controlador externo usado como um dispositivo de entrada, principalmente para jogos.

GHz (gigahertz)

Medida comum de um processador igual a um bilhão de ciclos por segundo.

Global System for Mobile Communications (GSM)

Rede mundial de celular.

GUI (graphical user interface, interface gráfica do usuário)

Interface que permite ao usuário navegar pelo sistema operacional usando ícones e menus.

GDI (Graphics Device Interface, interface de dispositivo gráfico)

Componente do Windows que gerencia como as imagens gráficas são transmitidas para os dispositivos de saída. O GDI converte as imagens para um bitmap que usa o computador em vez da impressora para transferir as imagens.

grayware

Spyware que se instala no computador sem ser solicitado e faz downloads de aplicativos adicionais sem a permissão do usuário.

half-duplex transmission (transmisso half-duplex)

Transmissão de dados que pode ser conduzida de duas formas, mas não ao mesmo tempo. Um telefone e um rádio bidirecional são exemplos desse tipo de transmissão.

handshaking sequence (seqüência de comunicação)

Uma série de breves comunicações que ocorrem entre dois modems. Isso estabelece a prontidão dos dois modems e dos computadores para iniciar a troca de dados.

handwriting recognition (reconhecimento de texto manuscrito)

Capacidade do computador, especialmente dispositivos móveis, para reconhecer letras e números escritos à mão e convertê-los para texto ASCII.

HDD (hard disk drive, unidade de disco rígido)

Dispositivo que armazena e recupera dados dos discos magnéticos que giram em velocidades bastante altas. O disco rígido, ou HDD, é a principal mídia de armazenamento em um computador.

HAL (Hardware Abstraction Layer, camada de abstração de hardware)

Biblioteca de drivers de hardware que estabelece comunicação entre o sistema operacional e o hardware instalado.

HCL (Hardware Compatibility List, lista de compatibilidade de hardware)

Utilitário que verifica se o hardware existente é compatível com um sistema operacional.

hardware firewall (firewall de hardware)

Dispositivo de hardware que filtra os pacotes de dados da rede antes que cheguem aos computadores e outros dispositivos na rede.

hardware

Componentes eletrônicos físicos que criam um sistema de computador.

Hayes-compatible command set (conjunto de comandos compatíveis com Hayes)

Conjunto de comandos AT utilizados pela maioria dos softwares de modem. Esse conjunto de comandos foi nomeado após a Hayes Microcomputer Products Company, empresa pioneira na definição de tais comandos.

headphone jack (entrada do fone de ouvido)

Soquete usado para conectar um dispositivo de saída áudio.

heat sink and fan assembly (dissipador de calor/ventoinha (cooler) )

Dispositivo que dissipa o calor dos componentes eletrônicos através da circulação do ar.

hex driver (chave de fenda hexagonal)

Chave usada para apertar porcas. Algumas vezes, é chamada de chave para porcas.

Hibernate/Standby indicator LED (LED indicador de hibernação/em espera)

Luz que mostra se o computador está no modo de espera (standby) ou em hibernação.

hierarchical star topology (topologia em estrela hierárquica)

Topologia em estrela estendida, onde um hub central é conectado pelo cabeamento vertical a outros hubs dependentes dele.

HDSL (High Data Rate DSL, linha digital de assinantes com alta taxa de dados)

Fornece uma largura de banda de 768 kbps em ambas as direções.

HDMI (High Definition Multimedia Interface, interface multimídia de alta definição)

Vídeo, plasma, LCD ou projetor DLP.

HKEY_

Designação inicial nos nomes de arquivo de inicialização do Windows Registry.

host

Sistema do computador em uma rede. Semelhante ao nó do termo, exceto pelo fato de que o host normalmente implica em um sistema de computador, considere que o nó geralmente se aplica a qualquer sistema de rede, inclusive servidores de acesso e roteadores.

hot-swappable interface (interface intercambiável)

Permite que os periféricos sejam alterados enquanto o sistema está sendo executado. O USB é um exemplo.

hub

1) Geralmente, um termo usado para descrever um dispositivo de Camada 1 que serve como o centro de uma rede de topologia estrela.
2) Dispositivo de hardware ou software que contém vários módulos de rede e equipamento de interligação de rede independentes, mas conectados. Os hubs podem ser ativos (quando eles repetem os sinais enviados por eles) ou passivos (quando eles não repetem, apenas dividem, os sinais enviados por eles).
3) Em Ethernet e IEEE 802.3, um repetidor multiportas Ethernet, algumas vezes citado como um centralizador.

HTML (Hypertext Markup Language, linguagem de marcação de hipertexto)

Linguagem de descrição da página usada por navegadores, como Windows Internet Explorer ou Mozilla Firefox.

HTTP (Hypertext Transfer Protocol, protocolo de transferência de hipertexto)

Define como é realizada a troca de arquivos na Internet.

I/O shield (blindagem de E/S)

Placa metálica de aterramento instalada na parte traseira do gabinete que permite que os conectores da placa-mãe sejam acessados pelo lado de fora do gabinete.

icon (ícone)

Imagem que representa um aplicativo ou um recurso.

IEEE 802.1

Uma especificação do IEEE que descreve um algoritmo que evita loops de bridging pela criação de uma spanning tree. O algoritmo foi inventado pela Digital Equipment Corporation. O algoritmo digital e o algoritmo IEEE 802.1 não são idênticos e nem compatíveis.

IEEE 802.12

Um padrão de rede local do IEEE que especifica a camada física e a subcamada MAC da camada de vínculo de dados. O IEEE 802.12 usa o esquema de prioridade para acesso aos meios físicos a 100 Mbps sobre uma variedade de meios físicos.

IEEE 802.2

Um protocolo de rede local do IEEE que especifica uma implementação da subcamada LLC da camada de vínculo de dados. O IEEE 802.2 processa erros, enquadramentos, controles de fluxos e a interface de serviço da camada de rede, Camada 3, interface de serviço. Usado em redes locais IEEE 802.3 e IEEE 802.5.

IEEE 802.3

Um protocolo de rede local do IEEE que especifica uma implementação da camada física e da subcamada MAC da camada de vínculo de dados. O IEEE 802.3 usa o acesso CSMA/CD a uma variedade de velocidades sobre uma variedade de meios físicos. Extensões do padrão IEEE 802.3 especificam implementações para Fast Ethernet. Variações físicas da especificação original do IEEE 802.3 incluem 10BASE2, 10BASE5, 10BASE-F, 10BASE-T e 10Broad36. Variações físicas da Fast Ethernet incluem 100BASE-T, 100BASE-T4 e 100BASE-X.

IEEE 802.3i

Uma variação física da especificação IEEE 802.3 original que especifica a utilização de sinalização do tipo Ethernet sobre meios de rede em pares trançados. O padrão define a velocidade de sinalização em 10 megabits por segundo usando um esquema de sinalização de banda base transmitida sobre cabos de par trançado usando uma topologia estrela ou estrela estendida.

IEEE 802.4

Um protocolo de rede local do IEEE que especifica uma implementação da camada física e da subcamada MAC da camada de vínculo de dados. O IEEE 802.4 usa acesso por passagem de token sobre uma topologia de barramento e baseia-se na arquitetura token bus para redes locais.

IEEE 802.5

Um protocolo de rede local IEEE que especifica uma implementação da camada física e da subcamada MAC da camada de vínculo de dados. O IEEE 802.5 usa acesso por passagem de token a 4 ou 16 Mbps sobre cabeamento STP e é semelhante ao IBM Token Ring.

IEEE 802.6

Uma especificação de MAN do IEEE baseada na tecnologia DQDB. O IEEE 802.6 suporta taxas de dados entre 1,5 e 155 Mbps.

impact printer (impressora de impacto)

Classe que inclui a impressora matricial e a impressora de margarida.

incremental backup (backup incremental)

Procedimento para fazer backup de todos os arquivos e pastas que foram criados ou modificados desde o último backup total ou normal.

IR (infrared, infravermelho)

Ondas eletromagnéticas cuja faixa de freqüências está acima das microondas, mas abaixo do espectro de luz visível. Os sistemas de rede local com base nesta tecnologia representam uma tecnologia emergente.

infrared port (porta de infravermelho)

Transceiver sem fio de linha de visão usado para a transmissão de dados.

infrared scatter (dispersão de infravermelho)

Sinal de infravermelho que encobre telhados e paredes. Dispositivos que podem estabelecer conexão sem uma linha de visão, mas com taxas de transferência de dados inferiores e pequenas distâncias.

inkjet printer (impressora jato de tinta)

Tipo de impressora que usa cartucho de tinta líquida, borrifando a tinta para formar uma imagem no papel.

input/output (I/O) (E/S (entrada/saída) )

Toda operação, programa ou dispositivo que transfere dados de ou para um computador.

input/output (I/O) address (endereço de E/S, entrada/saída)

Endereço de memória hexadecimal exclusivo associado a um dispositivo específico em um computador.

installation CD (CD de instalação)

CDs que incluem novo software com drivers e manuais. Além disso, podem incluir ferramentas de diagnósticos e software de teste (versão trial).

IM (instant messaging, mensagens instantâneas)

Método de comunicação com base em texto e em tempo real conduzido por uma rede entre dois ou mais usuários.

IEEE (Institute of Electrical and Electronics Engineers, instituto de engenharia elétrica e eletrônica)

Organização que acompanha o desenvolvimento dos padrões de comunicação e de rede.

insulation (isolante)

Material de alta resistência que impede o fluxo de corrente entre os condutores em um cabo.

ISDN (Integrated Services Digital Network, rede digital de serviços integrados)

Protocolo de comunicação, oferecido por empresas telefônicas, que permite que as redes telefônicas transportem dados, voz e outros tipos de tráfego.

interface

1) Conexão entre dois sistemas ou dispositivos.
2) Na terminologia de roteamento, uma conexão de rede.
3) Na telefonia, uma borda compartilhada definida por características de interconexões físicas, características de sinais e significados de sinais intercambiados.
4) A borda entre camadas adjacentes do modelo OSI.

IGP (Interior Gateway Protocol, protocolo de roteamento interno)

Protocolo de Internet usado para trocar informações de roteamento dentro de um sistema autônomo. Exemplos de IGPs comuns na Internet incluem EIGRP, OSPF e RIP.

IEC (International Electrotechnical Commission, comissão de eletrotécnica internacional)

Grupo industrial que escreve e distribui padrões para produtos e componentes elétricos.

ISO (International Organization for Standardization, organização internacional para padronização)

Uma organização internacional que define padrões para as redes. A ISO desenvolveu o modelo de referência OSI, um modelo de referência para redes largamente aceito.

Internet

Termo usado para referir-se à maior rede do mundo, conectando dezenas de milhares de redes ao redor do mundo.

IAB (Internet Architecture Board)

Junta de pesquisadores de rede que estudam questões pertinentes à arquitetura da Internet. Responsável pela indicação de uma variedade de grupos relacionados à Internet, como IANA, IESG e IRSG. A IAB é designada pelos curadores da ISOC.

ICMP (Internet Control Message Protocol, protocolo de mensagens de controle da Internet)

Usado para executar testes e solucionar problemas na rede, esse protocolo torna possível a emissão mensagens de erro e diagnósticos. As mensagens de eco do ICMP são usadas pelo utilitário ping para determinar se um dispositivo remoto está acessível.

IMAP (Internet Message Access Protocol, protocolo de acesso às mensagens de Internet)

Usado por cliente de e-mail local para sincronizar e recuperar e-mails de um servidor.

IP (Internet Protocol, protocolo de Internet)

Protocolo da camada de rede na pilha TCP/IP que oferece um serviços de rede sem conexão. O IP proporciona recursos para o endereçamento, especificação de tipo de serviço, fragmentação e remontagem e para segurança. Documentado no RFC 791.

ISP (Internet service provider, provedor de serviço na Internet)

Empresa que fornece o serviço de Internet a usuários domésticos, como empresas locais de telefonia ou serviços a cabo.

IPX/SPX (Internetwork Packet Exchange/Sequenced Packet Exchange, protocolo de intercâmbio de pacotes entre redes/intercâmbio de pacotes seqüenciados)

Usado pela Novell Netware; o IPX é capaz de estabelecer uma comunicação sem conexão e o SPX é a camada de transporte (Camada 7 do modelo de referência OSI).

IRQ (interrupt request, pedido de interrupção)

Pedido de um dispositivo de comunicação com a CPU.

Java

Linguagem de programação para applets a serem executados em um navegador da Web. Exemplos de applets incluem calculadora ou contador.

JavaScript

Linguagem de programação desenvolvida para interagir com o código fonte da linguagem HTML em sites interativos na Web. Dentre os exemplos estão o banner de rotacionamento e a janela popup.

jumper

Pontos de contato elétricos usados para definir uma unidade de disco rígido como Master ou Slave.

keyboard port (porta para teclado)

Soquete PS/2 usado para conectar um teclado externo.

keyboard (teclado)

Dispositivo de entrada com teclas multifuncionais.

KBps (kilobytes per second, kilobytes por segundo)

Medição da quantidade de dados que é transferida em uma conexão como, por exemplo, uma conexão de rede. Uma taxa de transferência de dados de 1 KBps é uma taxa de aproximadamente 1.000 bytes por segundo.

laptop battery (bateria do laptop)

Pilhas recarregáveis que proporcionam energia ao laptop.

laptop connector (conector do laptop)

Soquete usado para conectar o laptop a uma base multifuncional.

laptop keyboard (teclado do laptop)

Dispositivo de entrada que inclui teclas alfanuméricas, teclas de pontuação e teclas de função especial.

laptop latch (trava do laptop)

Trinco usado para abrir a tampa do laptop.

laptop

Computador de diâmetro total pequeno desenvolvido para ser portátil, mas que funciona igualmente a um computador de mesa. O hardware do laptop é proprietário e geralmente mais caro que o hardware do computador de mesa.

laser printer (impressora a laser)

Tipo de impressora que usa eletricidade estática e laser para formar uma imagem no papel.

latent image (imagem latente)

Em impressoras a laser, a imagem não é desenvolvida.

LCD monitor (monitor LCD)

Dispositivo de saída que transmite luz polarizada através de cristais líquidos para produzir imagens na tela.

LED (light-emitting diode, diodo emissor de luz)

Tipo de semicondutor que emite luz produzida pela energia elétrica. O LED indica se os componentes dentro do computador estão ligados.

LED indicador de status da bateria

Luz que indica a condição da bateria do laptop.

line of sight (linha de visão)

Característica de certos sistemas de transmissão, como laser, microondas e infravermelho, nos quais não podem existir obstruções dentro de uma linha direta entre o transmissor e o receptor.

line-in connector (conector de entrada de linha)

Soquete usado para conectar uma fonte de áudio.

LCD (liquid crystal display, tela de cristal líquido)

Tipo de tela de alta resolução, de peso leve que funciona bloqueando a luz em vez de criá-la.

LAN (local area network, rede de área local)

Rede de comunicação que cobre uma pequena área geográfica e está sob o controle de apenas um administrador.

local security policy (política de segurança local)

Combinação das configurações de segurança que definem a segurança do computador em que estão essas configurações.

logical drive (unidade lógica)

Seção na qual uma partição é dividida.

logical topology (topologia lógica)

Método real (anel ou barramento) pelo qual diferentes computadores e outros equipamentos em uma rede podem se comunicar com outro computador. Compare com topologia física.

loopback plug (plugue de retorno automático)

Ferramenta de diagnóstico que redireciona os sinais de volta à porta de origem que os transmitiu para solucionar problemas de conectividade.

MAC address (endereo MAC)

Endereço da camada de vínculo de dados padronizado necessária para cada porta ou dispositivo conectado a uma rede local. Outros dispositivos na rede utilizam esses endereços para localizar portas específicas na rede e para criar e atualizar tabelas de roteamento e estruturas de dados. Os endereços MAC possuem 6 bytes e são controlados pelo IEEE (Institute of Electrical and Electronics Engineers, instituto de engenharia elétrica e eletrônica). Também são conhecidos como endereço de hardware, endereço de camada MAC, endereço burnt-in ou endereço físico.

MDF (main distribution facility, instalação de distribuição principal)

Sala de comunicações principal de um prédio. Ponto central de uma topologia de rede em estrela na qual os patch panels, hubs e roteadores estão localizados.

malware

Termo criado pela junção das palavras malicioso (malicious) e software. Malware é um software que foi desenvolvido para infiltrar-se ou danificar o sistema do computador sem o consentimento do usuário.

MBR (Master Boot Record, registro de inicialização mestre)

Programa no primeiro setor de um disco rígido que começa o processo de inicialização. O MBR determina qual partição será usada para inicializar o sistema e transfere o controle para o setor de inicialização dessa partição, que continua o processo de inicialização. O MBR permite programas como o DOS para carregar RAM.

MSDS (Material Safety and Data Sheet, folha de dados de segurança)

Uma folha de dados que identifica materiais perigosos.

MTBF (mean time between failures, tempo médio entre falhas)

Tempo médio em que o dispositivo poderá funcionar sem falhar. As informações podem ser encontradas no manual ou no site do fabricante na Web.

media (meios físicos)

Plural de meio. Os vários ambientes físicos pelos quais passam os sinais de transmissão. Os meios de rede comumente usados incluem cabos de par trançado, coaxial e de fibra ótica e a atmosfera (por onde ocorrem as transmissões de microondas, laser e infravermelho).

MAC (Media Access Control, controle de acesso aos meios)

Camada mais inferior das duas subcamadas de vínculo de dados definida pelo IEEE (Institute of Electrical and Electronics Engineers, instituto de engenharia elétrica e eletrônica). A subcamada MAC lida com o acesso a meios compartilhados, como, por exemplo, se a contenção ou a passagem de token será usada. As regras de coordenação também usam os meios físicos em uma rede local.

media-handling options (opções de manuseio de mídias)

Opções pelas quais uma impressora lida com a mídia, incluindo orientação, tamanho e peso do papel.

megabit

1.048.576 bits (aproximadamente 1 milhão de bits).

Mbps (megabits per second, megabits por segundo)

Medição comum da quantidade de dados que é transferida em uma conexão em um segundo. Uma taxa de transferência de dados de 1 Mbps é uma taxa de aproximadamente 1 milhão de bits ou 1.000 kilobits por segundo.

MB (megabyte)

1.048.576 bytes (ou aproximadamente 1 milhão de bytes).

mesh grip (malha de arame)

Ferramenta conectada na extremidade de um cabo para ajudar a puxar o cabo.

mesh topology (topologia em malha)

Método de conexão de usuários que fornece caminhos alternativos para os dados. Se um caminho for cortado ou não puder ser utilizado, os dados poderão usar um caminho alternativo até seus destinos.

microphone (microfone)

Dispositivo de entrada de áudio.

microphone jack (entrada para microfone)

Soquete que serve para conectar um microfone usado para a entrada de áudio.

microwave (microondas)

Ondas eletromagnéticas no intervalo de 1 a 30 GHz. As redes que se baseia em microondas são uma tecnologia em evolução que ganha espaço devido à alta largura de banda e um custo relativamente baixo.

mobile processor (processador portátil)

CPU otimizada para usar menos energia permitindo que as baterias de laptops durem mais tempo.

modem port (porta do modem)

Tomada RJ-11 que conecta um computador a uma linha telefônica padrão. A porta do modem pode ser usada para conectar o computador com a Internet, para enviar e receber documentos por fax e para atender chamadas.

modem (modulator/demodulator, modulador/demodulador)

Dispositivo que converte os sinais digitais do computador em um formato que é enviado e recebido por uma linha telefônica analógica.

Molex power connector (conector de energia Molex)

Conector de energia para computador com quatro fios usado para conectar muitos dispositivos, como unidades ópticas e discos rígidos.

monitor

Dispositivo de vídeo que funciona com uma placa de vídeo instalada para apresentar os resultados a partir de um computador. A claridade de um monitor CRT baseia-se na largura de banda do vídeo, distância entre dois pontos, taxa de atualização e convergência.

motherboard (placa-mãe)

Principal placa de circuito impresso que conecta todos os componentes do computador, como CPU, BIOS, memória, interfaces de armazenamento em massa, portas seriais e paralelas, slots de expansão e controladores necessários para os dispositivos periféricos padrão.

mouse port (porta do mouse)

Soquete PS/2 usado para conectar um mouse externo.

MSCONFIG

Utilitário do Windows desenvolvido para auxiliar na solução de problemas do sistema operacional. Permite ao usuário editar aplicativos de inicialização e acessar os arquivos BOOT.INI, SYSTEM.INI e WIN.INI.

multimeter (multímetro)

Ferramenta de solução de problemas que mede a voltagem, a resistência e a corrente elétrica.

multimode (multímodo)

Fibra óptica com núcleo mais grosso que no modo simples. É mais fácil de criar, permite usar fontes luminosas mais simples, como LEDs, e funciona bem em pequenas distâncias. Esse tipo de fibra permite que as ondas de luz possam se dispersar em muitos caminhos enquanto são conduzidas pela fibra. 

multiprocessing (multiprocessamento)

Para permitir que os programas compartilhem duas ou mais CPUs.

MIME (Multipurpose Internet Mail Extensions, extensões multifunção para mensagens da Internet)

Padrão que estende o formato de e-mail para incluir texto no padrão ASCII, bem como outros formatos, por exemplo, imagens e documentos de processadores de texto. Normalmente, é usado em conjunto com o SMTP.

multitask (multitarefa)

Para executar dois ou mais aplicativos ao mesmo tempo.

multithread

Para dividir um programa em pequenas partes que podem ser carregadas conforme a necessidade do sistema operacional. O multiencadeamento (multithread) permite que os programas individuais sejam multitarefados.

multi-user (multiusuário)

Dois ou mais usuários executando programas e compartilhando dispositivos periféricos, como, por exemplo, uma impressora, ao mesmo tempo.

My Computer icon (ícone Meu computador)

Ícone da área de trabalho que fornece acesso às unidades instaladas e outras propriedades do computador.

NLQ (near letter quality, qualidade quase carta)

Qualidade de impressão melhor que a qualidade rascunho, mas não tão boa quando a qualidade carta.

needle-nose pliers (alicate de bico fino)

Ferramenta com pinças longas e finas que podem ser usadas para agarrar pequenos objetos.

NetView

Arquitetura de gerenciamento de rede da IBM e aplicativos relacionados. O NetView é um aplicativo VTAM (virtual telecommunications access method, método virtual de acesso às telecomunicações) usado para o gerenciamento de mainframes em redes SNA (Systems Network Architecture, arquitetura de rede de sistemas).

network (rede)

Grupo de dois ou mais dispositivos eletrônicos, como computadores, PDAs e smartphones que se comunicam uns com os outros para compartilhar dados e recursos.

NAP (Network Access Point, ponto de acesso à rede)

Ponto em que os provedores de acesso são interconectados.

network administration (administração da rede)

Tarefa de manutenção e atualização de uma rede particular realizada pelos administradores de rede.

NetBIOS (Network Basic Input/Output System, sistema de entrada/saída básico da rede)

API (Application programming interface, interface de programação de aplicativos) usada por aplicativos em uma rede local IBM para solicitar serviços a partir dos processos de rede em nível inferior. Esses serviços podem incluir estabelecimento e finalização de sessão e transferência de informações.

network cable (cabo de rede)

Meio físico usado para conectar dispositivos em conjunto para a comunicação.

network file services (serviços de arquivo de rede)

Permite que os documentos sejam compartilhados em rede para facilitar o desenvolvimento de um projeto.

network indicator LED (LED indicador de rede)

Luz que indica o status da conexão de rede. A luz verde indica a conectividade da rede. O outro LED indica o tráfego.

NIC (network interface card, placa de interface de rede)

É a interface do computador com a rede local. Geralmente, essa placa é inserida no computador em um slot de expansão e estabelece a conexão da rede.

network layer (camada da rede)

Camada 3 do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos). Essa camada fornece seleção de caminho e conectividade entre dois sistemas finais. A camada da rede é a camada em que ocorre o roteamento. De maneira geral, corresponde à camada de controle de caminho do modelo SNA (Systems Network Architecture, arquitetura de rede de sistemas). O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

NOS (network operating system, sistema operacional de rede)

Sistema operacional desenvolvido especificamente para fornecer recursos de rede adicionais.

network printer (impressora de rede)

Impressora conectada à rede de computadores configurada para ser compartilhada por vários usuários.

network server (servidor da rede)

Computador que fornece alguns serviços de rede, como, por exemplo, compartilhamento de arquivos, e pode lidar com vários usuários e vários trabalhos.

network topology (topologia de rede)

A maneira como computadores, impressoras e outros dispositivos são conectados.

networking media (meios da rede)

Material (cabo ou ar) pelo qual os sinais são enviados de um dispositivo de rede a outro.

NTFS (New Technology File System, nova tecnologia de sistema de arquivos)

Tipo de sistema de arquivos que fornece tolerância a falhas aprimorada em sistemas de arquivos tradicionais e também fornece segurança em nível de arquivo.

nibble

Metade de um byte ou quatro bits.

nó

1) Extremidade de uma conexão de rede ou junção comum a duas ou mais linhas em uma rede. Os nós podem ser processadores, controladores ou estações de trabalho. Eles também podem variar em roteamento e em outros recursos funcionais, podem ser interconectados por links e servir como pontos de controle na rede. Algumas vezes, o nó usado genericamente como referência para qualquer entidade que pode acessar uma rede e também é usado com freqüência de maneira intercambiável com dispositivos.
2) Em redes SNA (Systems Network Architecture, arquitetura de rede de sistemas), o componente básico de uma rede e o ponto no qual uma ou mais unidades funcionais conectam canais ou circuitos de dados.

noise (ruído)

Interferências, como EMI ou RFI, que causam sinais ruins e podem causar erros no sistema do computador.

non-bootable disk (disco não-inicializável)

Disco ausente ou danificado, ou um disco que não contém um ou mais arquivos de inicialização do sistema.

northbridge

Um dos dois chips no conjunto de chips lógico principal de uma placa-mãe do PC. Normalmente ele lida com a comunicação entre CPU, RAM, AGP, PCIe e chip do núcleo southbridge. Também chamado de MCH (Memory Controller Hub, hub controlador de memória).

NSLOOKUP

Comando que retorna o endereço IP a um determinado nome de host. Esse comando também pode reverter e localizar o nome de host de um endereço IP específico.

NTDETECT

Programa usado pelos sistemas com base em Intel para detectar o hardware instalado.

Num lock indicator LED (LED indicador de Num Lock)

Luz que indica se o teclado numérico está ativado ou desativado.

Ohm's Law (Lei de Ohm)

Relacionamento matemático entre corrente, resistência e voltagem, onde voltagem é igual à corrente multiplicada pela resistência.

operating system (sistema operacional)

Programa de software que executa tarefas gerais do sistema, como controlar RAM, priorizar processamento, controlar dispositivos de entrada e saída e gerenciar arquivos.

optical drive activity indicator LED (LED indicador de atividade da unidade óptica)

Luz que mostra a atividade da unidade.

optical drive (unidade óptica)

Unidade de disco que usa um laser para ler e/ou gravar CDs e DVDs.

pacote

Agrupamento lógico de informações que incluem um cabeçalho com informações de controle e, geralmente, dados do usuário. Em geral, os pacotes são mais usados para as unidades de dados da camada da rede. Os termos datagrama, quadro, mensagem e segmento também são usados para descrever agrupamentos de informações lógicas em várias camadas do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos) e em diversos círculos de tecnologia.

ping (Packet Internet Gopher, localizador de pacotes na Internet)

Simples utilitário de linha de comando que está incluído em muitas implementações de TCP/IP. O ping pode ser usado tanto com o nome de host quanto com o endereço IP para testar a conectividade. Determina se um endereço IP específico está acessível enviando uma solicitação de eco do ICMP para um computador de destino ou outro dispositivo de rede. Então, o dispositivo receptor retorna uma mensagem de resposta do eco do ICMP.

PDL (Page Description Language, linguagem de descrição de páginas)

Código que descreve o conteúdo de um documento em uma linguagem que pode ser compreendida pela impressora.

ppm (pages per minute, páginas por minuto)

Designação para medir a velocidade de uma impressora.

PATA (Parallel Advanced Technology Attachment)

Padrão para conexão de discos rígidos e unidades óticas em sistemas de computador e que usa tecnologia de sinalização paralela.

parallel ATA (PATA) data cable (cabo de dados (P)ATA paralelo)

Cabo interno que transfere dados entre a placa-mãe e a unidade de ATA.

parallel cable (cabo paralelo)

Cabo externo que conecta a porta paralela do computador a uma impressora ou outro dispositivo de comunicação paralelo. Também conhecido como cabo de impressora.

parallel port (porta paralela)

Soquete usado para conectar um dispositivo, como impressora ou scanner.

partition (partição)

Para dividir memória ou armazenamento em massa em seções isoladas ou lógicas. Quando um disco é particionado, o comportamento de cada partição será como unidade de disco separada.

PC Card

Placa de expansão usada em laptops para ajustar-se aos padrões PCMCIA.

P2P (peer-to-peer computing, computação ponto-a-ponto)

Cada dispositivo de rede executa ambas as partes de um aplicativo, tanto cliente quanto servidor. Também descreve a comunicação entre as implementações da mesma camada do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos) em dois dispositivos de rede diferentes.

PCMCIA (Personal Computer Memory Card International Association)

Associação comercial da indústria que define os padrões para a placa de expansão do laptop.

PDA (personal digital assistant, assistente digital pessoal)

Dispositivo portátil e independente com recursos de computação e comunicação.

Phillips-head screwdriver (chave de fenda Phillips)

Ferramenta usada para apertar ou soltar os parafusos em formato de cruz.

phishing

Tipo de spam intencional emitido para persuadir o destinatário a fornecer o remetente com informações que permitirão ao remetente acessar as informações pessoais do destinatário.

physical layer (camada física)

Layer 1 of the Open System Interconnection (OSI) reference model (Camada 1 do modelo de referência OSI, Open System Interconnection). A camada física define as especificações elétricas, mecânicas, funcionais e de procedimentos para ativar, manter e desativar o vínculo físico entre sistemas finais. Corresponde à camada de controle físico do modelo SNA (Systems Network Architecture, arquitetura de rede de sistemas). O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

physical topology (topologia física)

Layout físico dos componentes na rede.

piezoelectric (piezoelétrico)

Nas impressoras, uma placa carregada eletricamente altera o tamanho e a forma do injetor. A alteração no tamanho faz com que o injetor passe a agir como uma bomba. A ação de bombeamento força a tinta a sair pelo injetor e sobre o papel.

POTS (plain old telephone service, serviço telefônico comum)

Sistema telefônico comum que normalmente usa sinais analógicos para transmitir voz e dados. Algumas vezes é chamado de PSTN (Public Switched Telephone Network, rede de telefonia pública comutada).

platen (cilindro)

Cilindro da impressora matricial que aplica força contra o papel para evitar que ele deslize. Se for utilizado um papel de múltiplas cópias, a distância do cilindro poderá ser ajustada para a espessura do papel.

PnP (plug and play)

Tecnologia que permite ao computador configurar automaticamente os dispositivos que são conectados a ele.

POP (point of presence, ponto de presença)

O ponto de presença é o ponto de interconexão entre os recursos de comunicação fornecido pela companhia telefônica e o equipamento de distribuição principal do prédio.

port replicator (duplicador de porta)

Unidade de base fixa na qual um laptop é inserido e capaz de conectar dispositivos periféricos.

power adaptor (adaptador de energia)

Dispositivo que transforma corrente alternada em corrente contínua para fornecer eletricidade ao computador e carregar a bateria.

power button (botão liga/desliga)

Liga/desliga um dispositivo.

power cable (cabo de energia)

Cabo externo composto por condutores codificados em cores que transfere eletricidade ao computador e aos dispositivos elétricos conectados.

PLC (power line communication, comunicação da linha de energia)

Método de comunicação que usa fiações de distribuição de energia (grade elétrica local) para enviar e receber dados.

power on indicator LED (LED indicador de energia)

Luz que indica se o laptop está ligado ou desligado.

power supply (fonte de energia)

Converte a CA (alternating current, corrente alternada) em voltagens inferiores de DC (direct current, corrente contínua) que passa energia a todos os componentes do computador. As fontes de energia são mensuradas em Watts.

POST (power-on self-test, autoteste ao ligar-se)

Teste de diagnósticos de memória e hardware executado quando o sistema é ligado.

presentation layer (camada de apresentação)

Camada 6 do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos). Essa camada garante que a informação enviada pela camada de aplicação de um sistema poderá ser lida pela camada de aplicação de outro sistema. A camada de apresentação também lida com as estruturas de dados usadas pelos programas, negociando, assim, a sintaxe da transferência de dados da camada de aplicação. De maneira geral, corresponde à camada de serviços de apresentação do modelo SNA (Systems Network Architecture, arquitetura de rede de sistemas). O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

preventive maintenance policy (política de manutenção preventiva)

Programa detalhado que determina o tempo de manutenção, o tipo de manutenção desenvolvido e as especificações sobre como o plano de manutenção é transmitido.

preventive maintenance (manutenção preventiva)

Inspeção regular e sistemática, limpeza e substituição de peças, materiais e sistemas usados.

primary corona wire (fio de corona primário)

Dispositivo de voltagem que elimina a carga no tambor de impressão. Também chamado de grade ou cilindro de acondicionamento.

primary partition (partição primária)

Primeira partição em um disco rígido. Uma partição primária não pode ser subdividida em seções menores.

PRI (Primary Rate Interface, interface de taxa primária)

Interface ISDN (Integrated Services Digital Network, rede digital de serviços integrados, para o acesso de taxa primária) O acesso de taxa primária consiste em um canal D simples de 64 Kbps mais 23 canais B (T1) ou 30 (E1) para voz e dados. Compare com BRI.

print resolution (resolução de impressão)

Quantidade de pequenos pontos que pode ser impressa em um papel pelo cabeçote de impressão por polegada ao formar uma imagem.

PCL (Printer Control Language, linguagem de controle da impressora)

Desenvolvida pela Hewlett-Packard para permitir que os aplicativos de software possam estabelecer uma comunicação com a HP e com as impressoras a laser compatíveis com HP. Hoje, a PCL é um padrão industrial para muitos tipos de impressora.

printer driver (driver da impressora)

Software que deve ser instalado em um PC para que a impressora possa comunicar e coordenar o processo de impressão.

printer NIC (printer network interface card, placa de interface de rede)

Adaptador usado pela impressora para acessar os meios de rede.

printer queue (fila de impressão)

Área de armazenamento temporário dos trabalhos de impressão. Os trabalhos na fila serão alimentados para a impressora quando ela estiver pronta para o próximo trabalho.

printer-output options (opções de saída da impressora)

Determina a maneira como a tinta ou o toner são transferidos para o papel e inclui gerenciamento de cores, qualidade de impressão e velocidade.

protected mode (modo protegido)

Permite que os programas acessem mais de 1 MB da memória física e protege contra a utilização inadequada da memória, como programas que não pode executar um segmento de dados ou escrever em um segmento de código.

protocol (protocolo)

1) Descrição formal de um conjunto de regras e convenções que determina como os dispositivos de uma rede trocam informações.
2) Campo de um datagrama IP que indica o protocolo de camada superior (Camada 4) que envia o datagrama.

PDU (protocol data unit, unidade de dados de protocolo)

Uma unidade de dados especificada em um protocolo de uma camada do modelo OSI (Open System Interconnection, interconexão dos sistemas abertos). Por exemplo, a PDU da Camada 1 são bits ou o fluxo de dados, da Camada 2 são os enquadramentos, da Camada 3 é o pacote e da Camada 4 é o segmento.

proxy

Entidade que, para obter eficiência, representa outra entidade.

PSTN (Public Switched Telephone Network, rede de telefonia pública comutada)

Termo geral referente à variedade de redes e serviços telefônicos em todo o mundo. Algumas vezes é chamado de POTS (plain old telephone service, serviço telefônico comum).

RF (radio frequency, radiofreqüência)

Termo genérico referente às freqüências que correspondem a radiotransmissões. As redes de TV a cabo e de banda larga usam tecnologia de RF.

RFI (radio frequency interference, interferência de radiofreqüência)

Radiofreqüências que criam ruído que interfere nas informações transmitidas por cabeamento de cobre sem blindagem.

RAM (random access memory, memória de acesso aleatório)

Memória que armazena temporariamente os dados que serão processados pela CPU. Também chamado de memória física.

ROM (read only memory, memória apenas de leitura)

Memória que armazena definições de configuração pré-registradas e dados de maneira permanente em um chip que só pode ser lido. Esse tipo de memória retém seu conteúdo mesmo quando o chip não estiver recebendo energia.

RISC (Reduced Instruction Set Computer, computador com conjunto de instruções reduzido)

Arquitetura que usa um conjunto de instruções relativamente pequeno. Os chips RISC são desenvolvidos para executar essas instruções muito rapidamente.

RAID (Redundant Array of Independent Disks, matriz redundante de discos independentes)

Oferece tolerância a falhas impedindo a perda de dados caso ocorra uma falha no disco rígido em um servidor da rede. Também conhecido como Redundant Array of Inexpensive Disks, ou seja, matriz redundante de discos baratos.

Regedit

Aplicativo do Windows que permite aos usuários editar o registro.

registry (registro)

Amplo banco de dados do sistema usado pelo Windows para armazenar informações e configurações de hardware, software, usuários e preferências em um sistema.

RAS (remote access server, serviços de acesso remoto)

Servidor dedicado a usuários que precisam obter acesso aos arquivos e serviços de impressão na rede local de um local remoto.

RIS (remote installation services, serviços de instalação remota)

Capacidade de fazer download da instalação do sistema operacional Windows pela rede. Essa instalação pode ser solicitada pelo usuário ou pode ser forçada dentro do computador pelo administrador.

removable drive (unidade removível)

Unidade que pode ser removida de um computador para transportar dados.

resistance (r, resistência)

Medida, expressa em ohms, obtida pela oposição de um material a um fluxo de corrente.

resolution (resolução)

Quantidade de diferentes pixels em cada dimensão que pode ser exibida em uma tela de computador. Quanto maior a resolução, melhor a qualidade exibida na tela. Também conhecida como resolução da tela.

restore point (ponto de restauração)

Utilitário nos sistemas operacionais Windows Me, XP e Vista da Microsoft. Em caso de falhas no sistema, o ponto de restauração permite retornar os arquivos de sistema, chaves de registro e programas instalados para o estado anterior. Os dados de usuário não são afetados quando o ponto de restauração é executado.

ring topology (topologia em anel)

Topologia de rede que consiste em uma série de repetidores conectados uns aos outros por links de transmissão unidirecional para formar um loop único fechado. Cada estação da rede se conecta à rede em um repetidor. Mesmo tendo uma lógica de anel, as topologias de anel são organizadas com mais freqüência em forma de estrela de loop fechado.

router (roteador)

Dispositivo de camada de rede que usa uma ou mais medidas para determinar o caminho ideal pelo qual o tráfego da rede deve ser encaminhado. Os roteadores encaminham pacotes de uma rede para outra com base nas informações da camada de rede. Algumas vezes é chamado de gateway, embora a definição de gateway esteja se tornando cada vez mais ultrapassada.

RIP (Routing Information Protocol, protocolo de informações de roteamento)

IGP (Interior Gateway Protocol, protocolo de roteamento interno, fornecido com sistemas UNIX Berkeley Standard Distribution (BSD) ). O IGP mais comum da Internet. O RIP usa o contador de saltos como medida de roteamento.

safe mode (modo de segurança)

Opção de inicialização do sistema que carrega somente os dispositivos básicos que o Windows precisa executar. Esse processo é usado para a solução de problemas.

satellite communication (comunicação por satélite)

Utilização de satélites orbitais para transmitir dados entre múltiplas estações terrestres. A comunicação por satélite oferece alta largura de banda e capacidade de transmissão sob um custo que não está relacionado à distância entre as estações terrestres. Devido à altitude do satélite, esse tipo de comunicação pode estar sujeito a longos atrasos de propagação.

SCANDISK

Utilitário do Windows usado para examinar todos os arquivos em uma unidade.

security key fob (chave de segurança)

Pequeno sistema de rádio que se comunica com o computador por um breve intervalo. O computador deve receber o sinal da chave de segurança antes de aceitar os dados de nome de usuário e senha.

security keyhole (ponto da chave de segurança)

Ponto rígido no gabinete usado para conectar um cabo de segurança.

segment (segmento)

Parte de uma rede de computador na qual o dispositivo se comunica usando a camada física do modelo OSI. Hubs e repetidores são estendidos e tornam-se parte de um segmento de rede, enquanto os switches e roteadores definem e separam os segmentos de rede.

semicondutor

Material usado para criar chips de computador que podem ser usados como condutor ou como isolante, dependendo dos sinais de controle aplicados a ele. Os materiais mais comuns de um semicondutor são o silicone e o germânio, e estes materiais contêm outros materiais adicionados a eles que aumentam a condutividade.

serial cable (cabo serial)

Cabo externo que conecta a porta serial do computador a um dispositivo periférico.

serial data transfer (transferência de dados serial)

Movimento de bits simples de informação em um único ciclo.

serial port (porta serial)

Soquete usado para conectar um dispositivo, como mouse ou trackball.

serial transmission (transmissão serial)

Método de transmissão de dados no qual os bits de um caractere de dados são transmitidos seqüencialmente através de um único canal.

server (servidor)

Repositório para arquivos, ou outros recursos, que pode ser acessado e compartilhado pela rede por muitos usuários.

SLA (service level agreement, contrato de nível de serviço)

Contrato que define as expectativas entre uma organização e o fornecedor do serviço para fornecer um nível de suporte acordado.

session layer (camada de sessão)

Camada 5 do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos). Essa camada estabelece, gerencia e encerra sessões entre aplicações e gerencia troca de dados entre entidades de camadas de apresentação. O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

STP (shielded twisted pair, par trançado blindado)

Meio composto por dois pares de fios usados principalmente em redes token ring. O cabeamento do STP possui uma camada de isolamento blindado para reduzir a interferência eletromagnética (EMI). Compare com UTP.

shortcut (atalho)

Combinação no teclado que ativa um comando.

SMTP (Simple Mail Transfer Protocol, protocolo de transferência de correio simples)

Protocolo de e-mail usado por servidores para enviar mensagens de texto ASCII. Quando ampliado pelo protocolo MIME, o SMTP pode enviar e-mails com imagens e documentos. Algumas vezes, o SMTP é usado por clientes de e-mail para recuperar mensagens a partir de um servidor de e-mails. Entretanto, devido à capacidade limitada para as mensagens em fila ao fim de uma recepção, outros protocolos, como POP ou IMAP, são utilizados para receber e-mails.

simplex

Capacidade de transmissão de dados em apenas uma direção entre a estação de envio e uma estação de recebimento.

single-mode (modo simples)

Fibra óptica com núcleo muito fino. Usa alta energia a laser como fonte de luz e pode transmitir sinais a distâncias mais longas que o cabo de fibra óptica multimodo.

site survey (pesquisa de local)

Inspeção física do local que ajudará a determinar uma topologia de rede básica.

SCSI (Small Computer System Interface, interface de sistema de computadores de pequeno porte)

Padrão de interface paralela que suporta vários dispositivos no mesmo cabo e atinge taxas de transmissão de dados mais rápidas que os barramentos padrão.

Small Computer System Interface (cabo da SCSI (Small Computer System Interface, interface de sistema de computadores de pequeno porte) )

Cabo externo ou interno que conecta o controlador SCSI às portas SCSI de vários dispositivos internos e externos.

smart card (cartão inteligente)

Dispositivo do tamanho de um cartão de crédito que inclui processador e memória usado para armazenar informações e autenticar os usuários da rede. Os cartões inteligentes oferecem identificação dupla, pois o usuário deve ter tanto o cartão quanto a senha para acessar a rede.

software firewall (firewall de software)

Aplicativo em um computador que inspeciona e filtra os pacotes de dados.

solenoid (solenóide)

Bobina de fios que forma eletromagnéticos que acionam os pinos na impressora matricial.

solid-ink printers (impressoras de tinta sólida)

Impressora que usa bastões de tinta sólida em vez de toner ou cartuchos de tinta. Impressoras de tinta sólida produzem imagens de alta qualidade. Os bastões de tinta não são tóxicos e podem ser manipulados com segurança.

sound card (placa de som)

Placa de circuito integrado que aprimora os recursos de áudio do computador.

southbridge

Chip que implementa os recursos mais lentos da placa-mãe. É conectado à CPU pelo chip northbridge. Também é conhecido como ICH (Controlling Station, estação de controle) de E/S (Input/Output, entrada/saída).

spam

E-mail não solicitado.

speaker (alto-falante)

Dispositivo de saída de áudio.

spike (pico)

Aumento repentino na voltagem normalmente causado por relâmpagos.

spoof

Para obter acesso aos recursos dos dispositivos tentando passar por um computador confiável.

spooling

Processo de carregamento de documentos para um buffer (normalmente uma área no disco rígido) até que a impressora esteja pronta para imprimir os documentos.

spyware

Malware que monitora a atividade em um computador. O spyware envia as informações para a organização responsável pelo lançamento do spyware.

SPS (standby power supply, fonte de energia de reserva)

Reserva de energia que é ativada quando os níveis de voltagem caem abaixo do normal.

standoff (separador)

Obstáculo usado para separar fisicamente as peças.

star topology (topologia estrela)

Topologia de uma rede local na qual os pontos finais de uma rede são conectados a um switch central comum através de vínculos ponto-a-ponto. Uma topologia em anel que é organizada como uma estrela e implementa uma estrela de loop fechado unidirecional em vez de vínculos ponto-a-ponto.

SRAM (RAM estática)

Tipo de memória RAM que retém o conteúdo enquanto o dispositivo estiver ligado. A RAM estática não requer recuperação constante como a RAM dinâmica.

SRAM (static random access memory, memória de acesso aleatório estática)

Memória que retém dados enquanto houver voltagem aplicada. Usado principalmente como memória cache para a CPU.

stylus (caneta)

Utensílio de escrita. Geralmente usado como um método de entrada para as telas sensíveis ao toque de PDAs e tablets gráficos.

subnet mask (máscara de sub-rede)

Segundo grupo de números usado na configuração de um endereço IP em um dispositivo. A máscara de sub-rede é usada por dispositivos finais para determinar a parte da rede de um endereço IP.

subnetting (sub-rede)

Divisão lógica de uma rede. Ela fornece os meios para dividir uma rede e a máscara de sub-rede especifica como ela é subdividida.

surge (surto)

Toda voltagem que aumenta mais de 110% da voltagem normal transportada por uma linha de energia.

surge protector (protetor contra surto de voltagem)

Protetor que regula a voltagem que chega a um dispositivo.

surge suppressor (protetor anti-surto)

Dispositivo que verifica se a voltagem que chega a outro dispositivo está abaixo de um determinado nível.

S-video port (porta S-video)

Miniconector DIN de quatro pinos usado para emitir os sinais de vídeo a um dispositivo compatível. O S-video separa o brilho e a cor de um sinal de vídeo.

switch

1) Operação adicionada a um comando do DOS para modificar o resultado desse comando. 2) Um dispositivo de rede de Camada 2 também conhecido como ponte multiporta.

SDSL (Symmetric DSL, linha digital de assinantes simétrica)

Versão de um serviço DSL (Digital Subscriber Loop, loop digital de assinantes) que oferece a mesma velocidade para uploads e downloads.

Symmetric encryption (criptografia simétrica)

Criptografia que requer ambas as partes de uma conversa criptografada para usar uma chave de criptografia capaz de codificar e decodificar os dados. O remetente e o receptor devem usar a mesma chave ao mesmo tempo.

SYN flood (inundação SYN)

Abre aleatoriamente as portas TCP, bloqueando temporariamente o equipamento de rede ou os recursos do computador devido a uma grande quantidade de solicitações falsas, fazendo com que as sessões sejam negadas aos outros usuários. Veja também DoS (Denial of Service, ataques de negação de serviços).

System Restore (Restaurao do sistema)

Serviço do Windows XP que é executado em segundo plano e que permite ao usuário restaurar o sistema operacional para um período de tempo predefinido.

tape drive (unidade de fita)

Dispositivo usado para fazer backup de dados em uma unidade de servidor da rede.

Task Manager (Gerenciador de tarefas)

Exibe os aplicativos ativos e identifica aqueles que não estão respondendo, permitindo que eles sejam encerrados.

Taskbar (Barra de tarefas)

Utilitário do Microsoft Windows que representa graficamente os aplicativos abertos, o conteúdo do computador e outras informações. Também oferece acesso rápido a esses recursos.

TIA (Telecommunications Industry Association, associação da indústria de telecomunicações)

Organização que desenvolve os padrões relacionados às tecnologias das telecomunicações. Juntas, a TIA e a EIA (Electronic Industries Alliance, aliança das indústrias eletrônicas) formalizaram padrões como o EIA/TIA-232 para as características elétricas da transmissão de dados.

telnet

Aplicativo de acesso remoto usado para fornecer acesso terminal remoto entre os hosts em uma rede. Como ferramenta para a solução de problemas, o telnet pode verificar o software da camada de aplicação entre as estações de origem e de destino. Este é o mecanismo de teste mais completo disponível para o modelo OSI.

thermal paper (papel térmico)

Papel tratado quimicamente com qualidade de cera. Quando aquecido, torna-se preto. Muitos dos cabeçotes de impressão da impressora térmica são da largura do papel. O papel é fornecido em bobinas.

thermal printer (impressora térmica)

Uma impressora que marca o papel térmico especial aplicando calor nas áreas do papel que será escurecido para representar os caracteres.

Thicknet

Cabo coaxial que foi usado em redes antigas e operou em 10 megabits por segundo com um comprimento máximo de 500 metros. Também chamado de 10BASE5.

Thinnet

Cabo coaxial que foi usado em redes antigas e operou em 10 megabits por segundo com um comprimento máximo de 185 metros. Também chamado de 10BASE2.

three claw part retriever (alicate de três pontas)

Ferramenta usada para recuperar e manipular peças pequenas.

Token Ring network (rede Token Ring)

Usa uma topologia em anel e uma metodologia de passagem de token para criar uma transmissão de dados livre de colisões.

toner

Tinta em pó usada em impressoras a laser e fotocopiadoras para formar textos e imagens sobre o papel da impressora.

topology (topologia)

Layout físico real de uma rede ou, no caso de uma topologia lógica, o sinal ou os fluxos de dados em uma rede.

torx screwdriver (chave estrela)

Ferramenta usada para apertar ou soltar os parafusos que contém uma depressão em forma de estrela (seis pontas) na parte superior. Essa característica é encontrada principalmente nos parafusos para laptops.

touch screen

Monitor interativo em LCD ou CRT que detecta quando algo o pressiona, tocando na tela.

touchpad

Dispositivo sensível ao toque que controla o cursor.

tracert

Utilitário do Windows que traça a rota utilizada por um pacote a partir do computador de origem até o host de destino.

track

Círculo completo ao redor do disco rígido composto por grupos de setores de 512 bytes.

trackball

Esfera giratória que serve para controlar o cursor.

trackpoint

Almofada pequena e sensível ao toque que controla o cursor.

TCP (Transmission Control Protocol, protocolo de controle de transmissão)

Principal protocolo da Internet para a transmissão de dados. O TCP inclui recursos para estabelecer uma conexão ponta-a-ponta, detectar e recuperar erros e medir a taxa de fluxo de dados na rede. Muitos dos aplicativos padrão, como e-mail, navegador da Web, transferência de arquivos e Telnet, dependem dos serviços do TCP.

TCP/IP (Transmission Control Protocol/Internet Protocol, protocolo de controle de transmissão/protocolo da Internet)

Nome comum do conjunto de protocolos desenvolvidos pela U.S. DoD nos anos de 1970 para dar suporte à construção das redes mundiais. O TCP e o IP são os dois protocolos mais conhecidos do conjunto.

transport layer (camada de transporte)

Esta é a Camada 4 do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos). Ela é responsável pela comunicação confiável da rede entre os nós finais. A camada de transporte fornece mecanismos para o estabelecimento, a manutenção e o encerramento dos circuitos virtuais, detecção de falhas de transporte e recuperação, bem como o controle do fluxo de informações. Corresponde à camada de controle de transmissão do modelo SNA (Systems Network Architecture, arquitetura de rede de sistemas). O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

troubleshooting (solução de problemas)

Abordagem sistemática para localizar a causa de uma falha em um sistema de computador.

tweezers (pinças)

Ferramenta usada para recuperar e manipular peças pequenas.

twisted pair (par trançado)

Par de fios isolados e unidos em um padrão espiral regular que controla os efeitos do ruído elétrico. Os pares podem ser blindados ou não. O par trançado é comum nos aplicativos de telefonia e em redes de dados. Todos os cabos de par trançado de Categoria 3, Categoria 5, Categoria 5e e Categoria 6 contêm 4 pares trançados em um invólucro comum.

unattended installation (instalação autônoma)

Instalação personalizada de um sistema operacional com a mínima intervenção do usuário. O Windows executa instalações autônomas usando um arquivo de resposta chamado unattend.txt.

universal bay status indicator LED (LED indicador de status do compartimento universal)

Luz que indica que um dispositivo está instalado no compartimento do laptop.

Universal Serial Bus (USB) cable (cabo USB (Universal Serial Bus, barramento serial universal) )

Cabo externo que conecta a porta USB do computador a um dispositivo periférico.

Universal Serial Bus (USB) port (porta USB (Universal Serial Bus, barramento serial universal) )

Conexão externa bidirecional e intercambiável para conexão de cabos USB a dispositivos periféricos.

USB (Universal Serial Bus, barramento serial universal)

Interface de barramento serial externa para a conexão de vários dispositivos periféricos. O USB pode conectar até 127 dispositivos USB com taxa de transferência de até 480 Mbps e pode fornecer energia DC aos dispositivos conectados.

UNIX

Sistema operacional usado principalmente para executar e manter as redes de computador.

UTP (unshielded twisted pair, par trançado não blindado)

Quatro pares de fios usados em diversas redes. O UTP é classificado em categorias; quanto mais alta for a categoria, melhor será o desempenho e mais alta será a largura de banda. As categorias mais comuns são Categoria 3, Categoria 5, Categoria 5e, Categoria 6 e Categoria 6A.

UDP (User Datagram Protocol, protocolo de datagrama de usuário)

Serviço sem conexão para a passagem de dados com menos carga que o TCP e desenvolvido para velocidade. Os aplicativos de gerenciamento de rede, o sistema de arquivos de rede e o transporte de arquivos simples usam o UDP.

user interface (interface de usuário)

Parte do sistema operacional que permite ao usuário estabelecer comunicação com o computador. As interfaces de usuário fornecem uma interface de linha de comando (Command Line Interface) (texto) ou GUI (Graphical User Interface, interface gráfica do usuário).

user profile (perfil do usuário)

Configuração específica do usuário que está conectado com o computador.

ventilação

Permite expelir o ar quente do interior do dispositivo.

VDSL (very high data rate DSL, linha digital de assinantes com taxa de dados muito alta)

Transferência de dados de banda larga com larguras de banda de 13 Mbps a 52 Mbps.

video accelerator card (placa aceleradora de vídeo)

Placa de circuito integrado que contém processador e memória para aumentar a velocidade dos gráficos de vídeo. As placas de aceleração de vídeo são usadas principalmente para aplicativos em 3D e jogos.

video adapter (adaptador de vídeo)

Placa de circuito integrado que armazena dados digitais na VRAM e a converte em dados analógicos.

VGA (video graphics array, adaptador de vídeo)

Fornece vídeos analógicos para monitores analógicos. O conector é do subtipo D com 15 pinos.

video memory (memória de vídeo)

Memória de acesso aleatório dedicada em um adaptador gráfico de vídeo (vídeo RAM ou VRAM). Também é uma área da RAM do sistema compartilhada com a CPU.

video surveillance equipment (equipamento de vigilncia eletrnica, por vdeo)

Usado para gravar imagens e sons de uma atividade de monitoramento.

virtual memory (memória virtual)

Memória criada e controlada pelo sistema operacional que serve para manipular o espaço livre em disco imitando a RAM que está instalada no sistema.

VPN (virtual private network, rede privada virtual)

Sistema de criptografia que protege os dados enquanto eles são conduzidos pela Internet ou outra rede pública não segura.

virtual

Algo conceitual em vez de físico.

Virus Scan (verificação de vírus)

Utilitário que verifica a existência de vírus em todas as unidades de disco rígido e memória.

virus (vírus)

Em termos de computador, trata-se de uma parte maliciosa do software ou um código capaz de se copiar automaticamente e infectar um computador sem o conhecimento ou permissão do usuário. Alguns vírus são benignos e não afetam adversamente o computador, enquanto outros podem danificar ou excluir o sistema operacional e arquivos de dados.

VoIP (Voice over IP, voz sobre IP)

Método para transmitir chamadas telefônicas via Internet usando a tecnologia de comutação de pacotes.

V (voltage, voltagem)

Força que cria uma corrente através da movimentação de elétrons. Força eletromotriz ou diferença potencial expressa em volts.

volume control (controle de volume)

Botão que ajusta a saída de áudio.

warm boot (reinicialização)

Reinicia um computador que já estava ligado sem desligá-lo primeiro.

WYSIWYG (What You See Is What You Get, o formato exibido é o resultado final)

Resultado de um trabalho da impressora que corresponde ao o que o usuário vê na tela.

WAN (wide area network, rede de longa distância)

Rede de comunicação de dados que serve a usuários de uma ampla área geográfica e que, freqüentemente, usa dispositivos de transmissão oferecidos por transportadores comuns. Frame Relay, SMDS e X.25 são exemplos de WANs.

WPA (Wi-Fi Protected Access)

Padrão de segurança da tecnologia sem fio Wi-Fi. Fornece melhor criptografia e autenticação do que o sistema WEP anterior.

Wi-Fi

Marca originalmente licenciada pela Wi-Fi Alliance para definir a tecnologia incorporada de uma rede sem fio. A tecnologia Wi-Fi (Wireless Fidelity, fidelidade sem fio) tem como base as especificações IEEE 802.11.

Windows Explorer

Utilitário do Windows que representa graficamente a estrutura de gerenciamento de arquivos.

wire cutters (alicate para corte de fios)

Ferramenta usada para desencapar e cortar fios.

WEP (Wired Equivalent Privacy, privacidade equivalente de rede cabeada)

Padrão de segurança da primeira geração da tecnologia sem fio.

wireless connection (conexão sem fio)

Conexão com uma rede que usa sinais de rádio, tecnologia de infravermelho (laser) ou transmissões por satélite.

wireless indicator LED (LED indicador de rede sem fio)

Luz que indica a atividade da conexão de rede sem fio.

wireless network (rede sem fio)

Extensão de uma rede cabeada que usa os sinais de radiofreqüência (RF) para estabelecer conexão com pontos de acesso. Os sinais sem fio podem ser repetidos aos pontos de acesso adicionais, estendendo a distância da rede.

wireless NIC (placa de rede sem fio)

Placa de expansão que permite ao computador estabelecer conexão com um modem sem fio usando sinais de radiofreqüência.

WTLS (Wireless Transport Layer Security, segurança da camada de transporte sem fio)

Camada que fornece segurança para dispositivos móveis que usam WAP (Wireless Applications Protocol, protocolo para aplicações sem fio).

workgroup (grupo de trabalho)

Conjunto de estações de trabalho e servidores em uma rede local (LAN) desenvolvido para comunicar e trocar dados entre si.

ZIF socket (soquete ZIF, zero insertion force)

Soquete de chip que permite inserir ou remover um chip sem fazer uso de ferramentas ou de força. Sua aplicação é comum para chips delicados, como CPU.

1000BASE-T

Especificação Ethernet de Gigabit que usa UTP Cat5, 5e ou 6. Cada segmento de rede pode ter uma distância máxima de 328 pés (100 m) sem usar um repetidor. Também é conhecido como 801.3ab.

100BASE-TX

Especificação Fast Ethernet de banda básica de 100 Mbps que usa dois pares de fios UTP ou STP. Baseia-se no padrão IEEE 802.3.

100BASE-X

Especificação Fast Ethernet de banda básica de 100 Mbps que se refere aos padrões 100BASE-FX e 100BASE-TX para Fast Ethernet em cabeamento de cobre e fibra ótica. Baseia-se no padrão IEEE 802.3.

10BASE-T

Especificação Ethernet de banda básica de 10 Mbps que usa dois pares de cabeamento de par trançado Categoria 3, 4 ou 5. Um par para receber dados e o outro para transmitir dados. A 10BASE-T, que faz parte da especificação IEEE 802.3, tem uma distância limite de aproximadamente 328 pés (100 m) por segmento.
