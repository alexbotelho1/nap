NLQ (near letter quality, qualidade quase carta)

Qualidade de impressão melhor que a qualidade rascunho, mas não tão boa quando a qualidade carta.

needle-nose pliers (alicate de bico fino)

Ferramenta com pinças longas e finas que podem ser usadas para agarrar pequenos objetos.

NetView

Arquitetura de gerenciamento de rede da IBM e aplicativos relacionados. O NetView é um aplicativo VTAM (virtual telecommunications access method, método virtual de acesso às telecomunicações) usado para o gerenciamento de mainframes em redes SNA (Systems Network Architecture, arquitetura de rede de sistemas).

network (rede)

Grupo de dois ou mais dispositivos eletrônicos, como computadores, PDAs e smartphones que se comunicam uns com os outros para compartilhar dados e recursos.

NAP (Network Access Point, ponto de acesso à rede)

Ponto em que os provedores de acesso são interconectados.

network administration (administração da rede)

Tarefa de manutenção e atualização de uma rede particular realizada pelos administradores de rede.

NetBIOS (Network Basic Input/Output System, sistema de entrada/saída básico da rede)

API (Application programming interface, interface de programação de aplicativos) usada por aplicativos em uma rede local IBM para solicitar serviços a partir dos processos de rede em nível inferior. Esses serviços podem incluir estabelecimento e finalização de sessão e transferência de informações.

network cable (cabo de rede)

Meio físico usado para conectar dispositivos em conjunto para a comunicação.

network file services (serviços de arquivo de rede)

Permite que os documentos sejam compartilhados em rede para facilitar o desenvolvimento de um projeto.

network indicator LED (LED indicador de rede)

Luz que indica o status da conexão de rede. A luz verde indica a conectividade da rede. O outro LED indica o tráfego.

NIC (network interface card, placa de interface de rede)

É a interface do computador com a rede local. Geralmente, essa placa é inserida no computador em um slot de expansão e estabelece a conexão da rede.

network layer (camada da rede)

Camada 3 do modelo de referência OSI (Open System Interconnection, interconexão dos sistemas abertos). Essa camada fornece seleção de caminho e conectividade entre dois sistemas finais. A camada da rede é a camada em que ocorre o roteamento. De maneira geral, corresponde à camada de controle de caminho do modelo SNA (Systems Network Architecture, arquitetura de rede de sistemas). O modelo de referência OSI inclui camada de aplicação, camada de apresentação, camada de sessão, camada de transporte, camada de rede, camada de vínculo de dados e camada física.

NOS (network operating system, sistema operacional de rede)

Sistema operacional desenvolvido especificamente para fornecer recursos de rede adicionais.

network printer (impressora de rede)

Impressora conectada à rede de computadores configurada para ser compartilhada por vários usuários.

network server (servidor da rede)

Computador que fornece alguns serviços de rede, como, por exemplo, compartilhamento de arquivos, e pode lidar com vários usuários e vários trabalhos.

network topology (topologia de rede)

A maneira como computadores, impressoras e outros dispositivos são conectados.

networking media (meios da rede)

Material (cabo ou ar) pelo qual os sinais são enviados de um dispositivo de rede a outro.

NTFS (New Technology File System, nova tecnologia de sistema de arquivos)

Tipo de sistema de arquivos que fornece tolerância a falhas aprimorada em sistemas de arquivos tradicionais e também fornece segurança em nível de arquivo.

nibble

Metade de um byte ou quatro bits.

node (nó)

1) Extremidade de uma conexão de rede ou junção comum a duas ou mais linhas em uma rede. Os nós podem ser processadores, controladores ou estações de trabalho. Eles também podem variar em roteamento e em outros recursos funcionais, podem ser interconectados por links e servir como pontos de controle na rede. Algumas vezes, o nó usado genericamente como referência para qualquer entidade que pode acessar uma rede e também é usado com freqüência de maneira intercambiável com dispositivos.
2) Em redes SNA (Systems Network Architecture, arquitetura de rede de sistemas), o componente básico de uma rede e o ponto no qual uma ou mais unidades funcionais conectam canais ou circuitos de dados.

noise (ruído)

Interferências, como EMI ou RFI, que causam sinais ruins e podem causar erros no sistema do computador.

non-bootable disk (disco não-inicializável)

Disco ausente ou danificado, ou um disco que não contém um ou mais arquivos de inicialização do sistema.

northbridge

Um dos dois chips no conjunto de chips lógico principal de uma placa-mãe do PC. Normalmente ele lida com a comunicação entre CPU, RAM, AGP, PCIe e chip do núcleo southbridge. Também chamado de MCH (Memory Controller Hub, hub controlador de memória).

NSLOOKUP

Comando que retorna o endereço IP a um determinado nome de host. Esse comando também pode reverter e localizar o nome de host de um endereço IP específico.

NTDETECT

Programa usado pelos sistemas com base em Intel para detectar o hardware instalado.

Num lock indicator LED (LED indicador de Num Lock)

Luz que indica se o teclado numérico está ativado ou desativado.
