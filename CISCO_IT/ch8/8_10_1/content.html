8.10 Identificar nomes, finalidades e características de outras tecnologias usadas para estabelecer conectividade  
  8.10.1 Descrever as tecnologias telefônicas  

Existem várias soluções de WAN disponíveis para se conectar entre os sites ou à Internet. Os serviços de conexão WAN fornecem diferentes velocidades e níveis de serviço. Antes de se comprometer com qualquer tipo de conexão com a Internet, pesquise todos os serviços disponíveis para determinar a melhor solução para atender às necessidades do seu cliente.

Telefone analógico
Essa tecnologia usa linhas telefônicas de voz padrão. Esse tipo de serviço usa um modem para fazer uma ligação telefônica para outro modem em um local remoto, como um provedor. Existem duas grandes desvantagens de se usar a linha telefônica com um modem analógico. A primeira é que a linha telefônica não pode ser usada para chamadas de voz enquanto o modem estiver em uso. A segunda é a largura de banda limitada fornecida pelo serviço telefônico analógico. A largura de banda máxima usando um modem analógico é de 56 Kbps, mas, na realidade, é normalmente bem inferior a isso. Um model analógico não é uma boa solução para as demandas de redes movimentadas.

ISDN (Integrated Services Digital Network, rede digital de serviços integrados)
O próximo avanço em serviço de WAN é ISDN. O ISDN é um padrão para enviar voz, vídeo e dados por cabos normais de telefone. A tecnologia ISDN usa os fios de telefone como um serviço telefônico analógico. No entanto, o ISDN usa tecnologia digital para carregar os dados. Como ele usa tecnologia digital, o ISDN fornece uma tranasferência de dados de maior velocidade e voz com maior qualidade do que o serviço telefônico analógico tradicional.

Estes são os três serviços oferecidos por conexões digitais ISDN: BRI (Basic Rate Interface, interface de taxa básica), PRI (Primary Rate Interface, interface de taxa primária) e BISDN (Broadband ISDN, ISDN de banda larga). O ISDN usa dois tipos diferentes de canais de comunicação. O canal "B" é usado para carregar as informações – dados, voz ou vídeo – e o canal "D" é normalmente usado para controle e sinalização, mas pode ser usado para dados.

DSL (digital subscriber line, linha de assinante digital)
DSL é uma tecnologia contínua. “Contínua” significa que não há necessidade de discar a cada vez para se conectar à Internet. O DSL usa as linhas telefônicas de cobre existentes para fornecer comunicação de dados digital de alta velocidade entre os usuários finais e as empresas telefônicas. Diferentemente do ISDN, onde as comunicações de dados digitais substituem as comunicações de voz analógica, o DSL compartilha o fio de telefone com os sinais analógicos.

A empresa telefônica limita a largura de banda de voz analógica nas linhas. Esse limite permite que o DSL coloque os dados digitais no fio do telefone na parte não utilizada da largura de banda. Esse compartilhamento do fio de telefone permite que chamadas de voz sejam feitas enquanto o DSL está se conectando à Internet.

Existem duas considerações principais a serem feitas ao selecionar o DSL. DSL possui limitações de distância. As linhas telefônicas usadas com o DSL foram projetadas para carregar informações analógicas. Portanto, o comprimento no qual o sinal digital pode ser enviado é limitado e não pode passar por nenhum tipo de multiplexador usado com linhas telefônicas analógicas. A outra consideração é que as informações de voz e os dados carregados pelo DSL devem ser separados no local onde o cliente está. Um dispositivo chamado derivador separa a conexão com os telefones e a conexão com os dispositivos da rede local.

ADSL (asymmetric digital subscriber line, linha de assinante digital assimétrica)
ADSL é atualmente a tecnologia DSL mais comumente usada. O ADSL possui recursos de largura de banda diferentes em cada direção. Ele possui uma velocidade de downstream rápida – normalmente 1,5 Mbps. Downstream é o processo de transferir dados do servidor para o usuário final. Isso é benéfico aos usuários que estão baixando grandes quantidades de dados. A taxa de upload de alta velocidade do ADSL é mais lenta. O ADSL não funciona bem quando hospeda um servidor da Web ou FTP, pois os dois envolvem atividades de Internet com upload intenso.
