8.4 Descrever os componentes físicos de uma rede  
  8.4.1 Identificar nomes, finalidades e características dos dispositivos de rede  

Para tornar a transmissão de dados mais extensível e eficiente do que ocorre em uma rede simples ponto-a-ponto, os desenvolvedores de rede usam dispositivos de rede especializados, como hubs, switches, roteadores e pontos de acesso sem fio, para enviar dados entre os dispositivos.

Hubs
Os hubs, mostrados na Figura 1, são dispositivos que estendem o alcance de uma rede recebendo dados em uma única porta e, depois, gerando novamente os dados e os enviando a todas as outras portas. Esse processo significa que todo o tráfego de um dispositivo conectado ao hub é enviado a todos os outros dispositivos conectados ao hub toda vez que o hub transmite dados. Isso resulta em uma grande quantidade de tráfego de rede. Os hubs também são chamados de concentradores, pois servem como um ponto de conexão central para uma LAN.

Pontes e interruptores
Antes de serem transmitidos em uma rede, os arquivos são quebrados em pequenos pedaços chamados de pacotes. Esse processo permite a verificação de erros e uma retransmissão mais fácil caso o pacote seja perdido ou corrompido. As informações do endereço são adicionadas ao início e ao final dos pacotes antes de sua transmissão pela rede. O pacote, juntamente com as informações do endereço, é chamado de um quadro.

As LANs, geralmente, são divididas em seções chamadas de segmentos, similar ao modo como uma empresa é dividida em departamentos. Os limites dos segmentos podem ser definidos com o uso de uma ponte. Uma ponte é um dispositivo usado para filtrar o tráfego da rede entre os segmentos da LAN. As pontes mantêm um registro de todos os dispositivos em cada segmento ao qual a ponte está conectada. Quando as pontes recebem um quadro, o endereço de destino é examinado pela ponte para determinar se o quadro deve ser enviado para um segmento diferente, ou se deve ser derrubado. A ponte também ajuda a melhorar o fluxo de dados mantendo os quadros confinados somente ao segmento ao qual o quadro pertence.

Os switches, mostrados na Figura 2, são, às vezes, chamados de pontes multiportas. Uma ponte típica pode ter duas portas, vinculando dois segmentos da mesma rede. Um switch tem várias portas, dependendo de quantos segmentos de rede tiverem de ser vinculados. Um switch é um dispositivo mais sofisticado do que uma ponte. Um switch mantém uma tabela de endereços MAC para computadores que são conectados a cada porta. Quando um quadro chega a uma porta, o switch compara as informações do endereço no quadro com a tabela de endereços MAC. O switch, então, determina que porta deve ser usada para encaminhar o quadro.

Roteadores
Enquanto um switch conecta segmentos de uma rede, os roteadores, mostrados na Figura 3, são dispositivos que conectam redes inteiras umas às outras. Os switches usam endereços MAC para encaminhar um quadro dentro de uma única rede. Os roteadores usam endereços IP para encaminhar quadros para outras redes. Um roteador pode ser um computador com um software de rede especial instalado, ou pode ser um dispositivo criado por fabricantes de equipamentos de rede. Os roteadores contêm tabelas de endereços IP juntamente com rotas de destino otimizadas para outras redes.

Pontos de acesso sem fio
Os pontos de acesso sem fio, mostrados na Figura 4, fornecem um acesso de rede a dispositivos sem fio, como laptops e PDAs. O ponto de acesso sem fio usa ondas de rádio para se comunicar com rádios em computadores, PDAs e outros pontos de acesso sem fio. Um ponto de acesso tem um faixa limitada de cobertura. Redes grandes requerem vários pontos de acesso para fornecer uma cobertura sem fio adequada.

Dispositivos com várias funções
Há dispositivos de rede que executam mais de uma função. É mais conveniente adquirir e configurar um dispositivo que atenda a todas as suas necessidades do que adquirir um dispositivo separado para cada função. Isso é especialmente verdade para o usuário doméstico. Em sua casa, você poderia adquirir um dispositivo com várias funções em vez de um switch, um roteador e um pondo de acesso sem fio.
