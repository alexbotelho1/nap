8.5 Descrever as topologias e arquiteturas da LAN  

A maioria dos computadores com os quais você trabalha fará parte de uma rede. Topologias e arquiteturas são blocos de criação para o desenvolvimento de uma rede de computadores. Embora você ainda não possa criar uma rede de computadores, terá de compreender como esses blocos são projetados para poder trabalhar em computadores que façam parte de uma rede.

Existem dois tipos de topologias de LAN: física e lógica. Uma topologia física, mostrada na Figura 1, é o layout físico dos componentes na rede. Uma topologia lógica, mostrada na Figura 2, determina como os hosts se comunicam por meio de uma mídia, como um cabo ou ondas de transmissão. As topologias são comumente representadas como diagramas de rede.

Uma arquitetura de LAN é criada ao redor de uma topologia. A arquitetura de LAN compreende todos os componentes que compõem a estrutura de um sistema de comunicação. Esses componentes incluem o hardware, o software, os protocolos e a seqüência das operações.
