8.2 Descrever os tipos de redes  

As redes de dados continuam a se desenvolver quando à complexidade, ao uso e design. Para se comunicar sobre as redes, diferentes tipos de redes recebem diferentes nomes descritivos. Uma rede de computadores é identificada pelas seguintes características específicas:

  • A área que ela serve
  • Como os dados são armazenados
  • Como os recursos são gerenciados
  • Como a rede é organizada
  • O tipo dos dispositivos de rede usados
  • O tipo de mídia usado para conectar os dispositivos

Após a conclusão desta seção, você estará apto a:

  • Descrever uma LAN.
  • Descrever uma WAN.
  • Descrever uma WLAN.
  • Explicar as redes ponto-a-ponto.
  • Explicar as redes cliente/servidor.
