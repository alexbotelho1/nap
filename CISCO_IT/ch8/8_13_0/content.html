8.13 Resumo  

Este capítulo apresentou a você os fundamentos de rede, os benefícios de possuir uma rede e as maneiras de se conectar computadores a uma rede. Os diferentes aspectos de solucionar problemas de rede foram discutidos com exemplos de como analisar e implementar soluções simples. É importante lembrar os seguintes conceitos deste capítulo:

  • Uma rede de computadores é composta de dois ou mais computadores que compartilham dados e recursos.
  • Uma LAN (Local Area Network, rede local) refere-se a um grupo de computadores interconectados que estão sob o mesmo controle administrativo.
  • Uma WAN (Wide Area Network, rede de longa distância) é uma rede que conecta LANs em locais geograficamente separados.
  • Em uma rede ponto-a-ponto, os dispositivos são conectados diretamente um ao outro. Uma rede ponto-a-ponto é fácil de instalar e nenhum equipamento adicional ou administrador dedicado é necessário. Os usuários controlam seus próprios recursos e uma rede funciona melhor com um pequeno número de computadores. Uma rede cliente/servidor usa um sistema dedicado que funciona como o servidor. O servidor responde às solicitações feitas pelos usuários ou clientes conectados à rede.
  • Uma LAN usa uma conexão direta de um computador a outro. É adequada para uma área pequena, como em uma casa, um prédio ou uma escola. Uma WAN usa linhas de comunicações seriais, ponto-a-ponto ou ponto-a-multiponto para comunicar em distâncias maiores. Uma WLAN usa tecnologia sem fio para conectar dispositivos.
  • A topologia de rede define a maneira pela qual computadores, impressoras e outros dispositivos são conectados. A topologia física descreve o layout do fio e dos dispositivos, bem como os caminhos usados pelas transmissões de dados. A topologia lógica é o caminho pelo qual os sinais viajam de um ponto a outro. As topologias incluem barramento, estrela, anel e malha.
  • Os dispositivos de rede são usados para conectar computadores e periféricos para que eles possam se comunicar. Eles incluem hubs, pontes, comutadores, roteadores e dispositivos com várias funções. O tipo de dispositivo implementado depende do tipo de rede.
  • Os meios da rede podem ser definidos como o meio pelo qual os sinais, ou dados, são enviados de um computador a outro. Os sinais podem ser transmitidos por cabo ou sem fio. Os tipos de meios discutidos foram cabeamento coaxial, par trançado, fibra óptica e freqüências de rádio.
  • A arquitetura Ethernet é agora o tipo mais popular de arquitetura de LAN. A arquitetura se refere à estrutura global de um computador ou sistema de comunicações. Ela determina os recursos e as limitações do sistema. A arquitetura Ethernet é baseada no padrão IEEE 802.3. O padrão IEEE 802.3 especifica que uma rede implemente o método de controle de acesso CSMA/CD.
  • O modelo de referência OSI é uma estrutura padrão da indústria usada para dividir as funções da rede em sete camadas distintas. Essas camadas incluem aplicativo, apresentação, sessão, transporte, rede, vínculo de dados e física. É importante entender a finalidade de cada camada.
  • O conjunto de protocolos TCP/IP tornou-se o padrão dominante para a Internet. O TCP/IP representa um conjunto de padrões públicos que especificam como será a troca dos pacotes de informações entre os computadores de uma ou mais redes.
  • Uma NIC é um dispositivo que se conecta a uma placa-mãe e fornece portas para as conexões de cabo de rede. É a interface do computador com a LAN.
  • Um modem é um dispositivo eletrônico usado para comunicações de computador por meio de linhas telefônicas. Ele permite transferência de dados entre um computador e outro. O modem converte dados orientados por bytes em fluxos de bits seriais. Todos os modems requerem software para controlar a sessão de comunicação. O conjunto de comandos que a maioria dos softwares de modem usa é conhecido como o conjunto de comandos compatíveis com Hayes.
  • Os três métodos de transmissão para enviar sinais por canais de dados são simplex, half-duplex e full-duplex. A tecnologia de rede full-duplex aumenta o desempenho porque os dados podem ser enviados e recebidos ao mesmo tempo. DSL, modem a cabo bidirecional e outras tecnologias de banda larga operam no modo full-duplex.
  • Os dispositivos de rede e os meios, como componentes do computador, devem ser mantidos. É importante limpar os equipamentos regularmente e usar uma abordagem proativa para evitar problemas. Conserte ou substitua equipamentos quebrados para evitar paralisações por manutenção.
  • Ao solucionar problemas de rede, ouça o que o cliente tem a dizer para formular perguntas abertas e fechadas que ajudarão a determinar onde começar a corrigir o problema. Verifique problemas óbvios e tente aplicar soluções rápidas antes de expandir o processo de solução de problemas.
