16.2 Selecionar os componentes de segurança com base nas necessidades do cliente  
  16.2.2 Descrever e comparar os dispositivos de controle de acesso  

Os dados e equipamentos de computador podem ser protegidos por meio de técnicas de proteção sobrepostas a fim de evitar o acesso não autorizado a dados confidenciais. Como exemplo de proteção sobreposta, podemos citar o uso de duas técnicas diferentes para proteger um bem. Essa medida é conhecida como segurança two-factor ou dupla, como mostra a Figura 1. No momento de decidir sobre um programa de segurança, é preciso equilibrar o custo da implementação com o valor dos dados ou equipamentos a serem protegidos.

Segurança física
Utilize o hardware de segurança para ajudar a evitar violações de segurança e perda de dados ou equipamentos. As medidas de controle de acesso por segurança física incluem o seguinte:

  • Trava – O dispositivo mais comum para proteger áreas físicas. Se uma chave for perdida, todas as travas de chaves idênticas devem ser modificadas.
  • Conduíte – Proteção para a mídia de infra-estrutura contra danos e acesso não autorizado.
  • Chave cartão – Ferramenta usada para proteger áreas físicas. Se uma chave cartão for perdida ou roubada, somente o cartão desaparecido precisa ser desativado. O sistema de chave cartão é mais caro do que as travas de segurança.
  • Equipamento de vídeo – Grava imagens e sons para atividade de monitoramento. Os dados gravados devem ser monitorados para que se detectem problemas.
  • Guarda de segurança – Controla o acesso à entrada do estabelecimento e monitora a atividade dentro do local.

O equipamento de rede deve ser preparado em áreas seguras. Todo o cabeamento deve estar recolhido dentro de conduítes ou passar por dentro das paredes para evitar adulterações ou acesso não autorizado. As saídas de rede que não estão em uso devem ser desativadas. Se o equipamento de rede for danificado ou roubado, pode ser necessário recusar serviço a alguns usuários da rede.

A política de segurança deve especificar o nível de segurança necessário na organização. Os dispositivos biométricos, que captam informações físicas sobre os usuários, são ideais para o uso em áreas extremamente seguras. No entanto, esse tipo de solução seria caro demais para a maioria das pequenas empresas.

Segurança dos dados
É possível proteger os dados usando dispositivos de segurança de dados para autenticar o acesso dos funcionários. A identificação two-factor é um dos métodos usados para aumentar a segurança. Os funcionários devem usar uma senha e um dispositivo de segurança de dados semelhante aos descritos aqui para acessar os dados:

  • Smart card (cartão inteligente) – Dispositivo capaz de armazenar dados com segurança. A memória interna é um ICC (integrated circuit chip, chip de circuito integrado) embutido que conecta a um leitor diretamente ou por meio de uma conexão sem fio. Os smart cards são usados para diversas finalidades no mundo inteiro, como crachás de identificação seguros, dispositivos de autenticação on-line e pagamentos seguros com cartão de crédito.
  • Security key fob (chave de segurança) – Pequeno dispositivo semelhante a um chaveiro. Possui um pequeno sistema de rádio que se comunica com o computador por um breve intervalo. O chaveiro é pequeno o suficiente para que seja preso junto com chaves comuns. O computador deve receber o sinal da chave de segurança antes de aceitar o nome de usuário e a senha.
  • Dispositivo biométrico – Registra características físicas do usuário, tais como digitais ou padrões da íris. O usuário obtém acesso se as características corresponderem com o banco de dados e, então, as informações corretas de login são fornecidas.

O nível de segurança necessário para o cliente determina quais dispositivos devem ser escolhidos para manter dados e equipamentos seguros.
