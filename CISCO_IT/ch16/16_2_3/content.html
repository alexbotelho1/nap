  16.2.3 Descrever e comparar os tipos de firewall  

Os firewalls de hardware e de software protegem os dados e os equipamentos das redes contra acesso não autorizado. O firewall deve ser usado junto com o software de segurança.

Os firewalls de hardware e de software possuem diversos modos de filtro do tráfego de dados na rede:

  • Filtro de pacotes – Um conjunto de regras que permitem ou proíbem o tráfego com base em critérios como os endereços, protocolos ou portas IP utilizados.
  • Firewall de proxy – Um firewall que inspeciona todo o tráfego e permite ou proíbe os pacotes com base em regras configuradas. O proxy funciona como um gateway que protege os computadores dentro da rede.
  • Inspeção de estado de pacotes – Um firewall que monitora o estado das conexões de rede que trafegam pelo firewall. Os pacotes que não fazem parte de uma conexão conhecida não têm permissão de retorno pelo firewall.

Firewall de hardware
O firewall de hardware é um componente físico de filtro que inspeciona os pacotes de dados da rede antes que cheguem aos computadores e outros dispositivos na rede. Os firewalls de hardware costumam ser instalados em roteadores. O firewall de hardware é uma unidade independente que não usa os recursos dos computadores que está protegendo, por isso não há impacto no desempenho do processamento.

Firewall de software
O firewall de software é um aplicativo em um computador que inspeciona e filtra os pacotes de dados. O Windows Firewall é um exemplo de firewall de software que é incluído no sistema operacional Windows. O firewall de software usa os recursos do computador, o que resulta em diminuição do desempenho para o usuário.

Considere os itens descritos na Figura 1 quando selecionar um firewall.

OBSERVAÇÃO: em uma rede segura, se o desempenho do computador for irrelevante, é recomendável ativar um firewall interno de sistema operacional para garantir uma segurança adicional. Alguns aplicativos podem não funcionar adequadamente se o firewall não estiver corretamente configurado para eles.
