16.4 Desenvolver a manutenção preventiva em segurança  
  16.4.3 Explicar os procedimentos de backup dos dados, acessar os backups e proteger a mídia de backup física  

O backup de dados armazena uma cópia das informações de um computador em uma mídia de backup removível que pode ser mantida em local seguro. Se o hardware do computador falhar, o backup de dados pode ser restaurado para dar continuidade ao processamento.

Os backups de dados devem ser realizados regularmente. O backup de dados mais atual geralmente é armazenado externamente para proteger a mídia de backup caso alguma coisa aconteça nas instalações principais. A mídia de backup, em geral, é reutilizada visando à economia nas despesas com mídias. Sempre cumpra as diretrizes de rodízio de mídia de sua organização.

As operações de backup podem ser realizadas por linha de comando ou a partir de um arquivo batch usando o comando NTBACKUP. Os parâmetros padrão para o NTBACKUP serão iguais aos definidos no utilitário de backup do Windows. Qualquer opção que você desejar substituir deve ser incluída na linha de comando. O comando NTBACKUP não pode ser usado para restaurar arquivos.

Uma combinação de tipos de backup, como mostra a Figura 1, permite que os dados sejam submetidos ao backup de modo eficiente. O backup completo é a cópia de todos os arquivos da unidade. Um procedimento de backup incremental faz o backup apenas dos arquivos criados ou modificados desde o último backup normal ou incremental. Ele assinala os arquivos indicando que foram submetidos ao backup. O backup diferencial copia os arquivos criados ou modificados desde o último backup normal ou incremental, mas não assinala os arquivos indicando que foram submetidos ao backup. A realização do backup de dados pode ser demorada. Por isso, é preferível fazê-lo quando o tráfego da rede for baixo. Há ainda outros tipos de backups como o backup diário e o backup de cópia, que não assinalam os arquivos indicando que foram submetidos ao backup.

A mídia de backup de dados é tão importante quanto os dados do computador. É aconselhável armazená-la em um local de armazenamento externo com regulação de temperatura e com a devida segurança física. É importante que os backups estejam disponíveis imediatamente para o caso de emergência.
