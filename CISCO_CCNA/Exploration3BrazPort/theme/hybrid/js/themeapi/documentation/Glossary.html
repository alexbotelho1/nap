addTerm

anulação addTerm( t,  d,  i,  s,  sa)
    Adiciona um objeto de termo aos arranjos de termo e, se houver o conjunto, adiciona esse termo ao arranjo de termos do conjunto
    Parâmetros:
      t - termo
      d - definição
      i - ID do termo
      s - ID do conjunto
      sa - termos consulte também

getTermByID

GlossaryTerm getTermByID( i)

    Retorna:
      Referência ao objeto GlossaryTerm

setData

anulação setData( d)
    Define dados no objeto usando uma cadeia de caracteres delimitada enviada por meio de jshook api. A cadeia de caracteres é dividida, e cada entrada é executada por meio de eval().
    Parâmetros:
      d - cadeia de caracteres delimitada com comandos js
