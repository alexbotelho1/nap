    Parâmetros:

recCommand

anulação  recCommand( c)
    Recebe resposta do item de memória flash com base no comando enviado. Se o comando enviado não for válido e retornar erros. Se o comando fosse válido, os dados seriam disponibilizados dentro da memória flash como uma variável, recuperada e acionada com base no tipo de comando.
    Parâmetros:
      c - comando recebido

sendCommand

anulação  sendCommand( c, p)
    Envia o comando e as propriedades para o item de memória flash
    Parâmetros:
      c - comando a ser enviado
      p - propriedades a serem enviadas

anulação  setReady(s)
    Método definiu o conteúdo como pronto
