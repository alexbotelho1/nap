Detalhes do método

getCurrent

anulação getCurrent()
    envia o comando para obter as informações de página atuais

gotoPage

anulação gotoPage( p)
    Envia o comando de ir para a página informando o número LI fornecido
    Parâmetros:
      p - "." número LI delimitado para a página a ser navegado até, ex "1.2.3.4"
    Consulte:
      - previousPage()
      - nextPage()

nextPage

anulação nextPage()
    Envia o comando de próxima página
    Consulte:
      - previousPage()
      - gotoPage()

previousPage

anulação previousPage()
    Envia o comando de página anterior
    Consulte:
      - nextPage()
      - gotoPage()

setCurrentChapterByID

anulação setCurrentChapterByID( i)
    Define o capítulo de uma ID fornecida
    Parâmetros:
      i - ID do capítulo

setCurrentPageByID

anulação setCurrentPageByID( i)
    Define a página a partir de uma ID fornecida e, se a página não tiver um texto, a chamada será feita para obtê-lo.
    Parâmetros:
      i - ID da página

setCurrentSectionByID

anulação setCurrentSectionByID( i)
    Define a seção de uma ID fornecida
    Parâmetros:
      i - ID da seção

setCurrentTopicByID

anulação setCurrentTopicByID( i)
    Define o tópico de uma ID fornecida
    Parâmetros:
      i - ID do tópico

setData

anulação setData( d)
    Define dados no objeto usando uma cadeia de caracteres delimitada enviada por meio de jshook api. A cadeia de caracteres é dividida, e cada entrada é executada por meio de eval().
    Parâmetros:
      d - cadeia de caracteres delimitada por comandos js

toggle

anulação toggle()
    Alterna do idioma atual para o idioma secundário e atualiza os dados e o tema
